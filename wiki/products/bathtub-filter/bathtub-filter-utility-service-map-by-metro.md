# Bathtub Filter — Top 100 美国城市 utility 消毒剂 / 硬度 速查表

## 为什么有这页

`water-jurisdiction-demand-map` 已经声明 KES V1 scope = **NA free-chlorine 地区**。但**没有一张表把这个 scope 翻译成具体可执行的 metro 列表**——意味着 marketing / paid acquisition / retail rollout 没法按 zip 切。

这页解决：

> **KES V1 该在哪些 metro 优先 launch？哪些 metro 必须排除（chloramine-heavy）？哪些 metro 因为硬水高需要额外免责声明？**

## 1. 一句话结论

按 top 100 美国城市抽样：

- **约 60% 用 chloramine**（包括 LA / Houston / Dallas / DC / Philly / Boston / SF / San Diego / Miami 等高人口市场）— **不在 V1 scope**
- **约 40% 用 free chlorine**（包括 NYC / Chicago / Atlanta / Phoenix / Las Vegas / Seattle / Sacramento / 多数 FL 市场）— **V1 scope**
- **西南部（NV/AZ/UT/南 CA）硬水叠加**——即便是 chlorine 市场，KES copy 也必须主动免责"不软化硬水"，否则触发"骗钱"型差评

**关键含义：KES V1 的 addressable TAM 比泛指"美国"小约 60%。** 必须从一开始按 zip 圈定 paid 渠道、按州划分 wholesale 推广。

## 3. KES V1 metro 优先级矩阵（基于消毒剂 + 硬水 + 人口）

| 等级 | 标准 | 推荐 metro |
|---|---|---|
| **🥇 Tier A — 必投** | free chlorine + 中/软水 + 1M+ metro pop | **NYC, Chicago, Atlanta, Jacksonville, Memphis, Charlotte, Detroit, Pittsburgh, Seattle, Portland** |
| 🥈 Tier B — 投但 copy 加免责 | free chlorine + **硬水** + 1M+ metro | **Phoenix, Las Vegas, San Antonio, Tucson, Mesa**（必须主动声明"不软化硬水"） |
| 🥉 Tier C — 二期评估 | free chlorine 但 metro pop 较小 | Albuquerque, Honolulu, Baton Rouge, Bakersfield, Lubbock, Spokane, Toledo, Sacramento |
| ⛔ Tier X — V1 排除 | chloramine | 见 §2.2 完整列表 |
| ⚠️ Tier Y — 过渡期警戒 | chlorine→chloramine transition 中 | **Los Angeles**（2027–2029 漂移）、SLO County 反向（chloramine→chlorine） |

## 4. Watch list（消毒剂会变的市场）

**正在 / 计划过渡 chlorine→chloramine（V1 scope 在缩水）：**
- **Los Angeles** — LADWP 99th St plant、North Hollywood Ammoniation Station、Mission Wells Chloramination Station 建设中，2027–2029 全城完成
- 各 CA 都市区 historical 多数已经过渡，剩余 chlorine 市场可能继续转换

**反向 chloramine→chlorine（V1 scope 在扩张）：**
- San Luis Obispo County (Five Cities Area, CA) — 已宣布转 free chlorine
- 多数反向转换是临时性（用于管网消毒维护），不是永久 scope 变化

**KES 含义：** KES marketing dashboard 必须每季度复查 §2 表 + watch list。**LA 不要做长期投放假设**，把 LA 视为 12–24 个月窗口资源。

## 5. 与 V1 product copy 的对接

### 5.1 Tier A 城市（free chlorine + 软/中水）
- 标准 KES copy 直接可用
- emphasis: chlorine reduction（最强 perceived value）

### 5.2 Tier B 城市（free chlorine + 硬水）
- **必须**在 product page、Amazon listing、packaging 加免责段：
  > "本产品针对自来水中的余氯设计。如果你所在地区的水硬度高（例如 Phoenix / Las Vegas / Tucson），本产品**不会软化你的水**，建议同时使用软水机或软水伴侣。"
- 否则触发"我以为它能软化我的水"型差评（参考 [[bathtub-filter-marketplace-negative-review-signals]] 的 expectation gap 簇）

### 5.3 Tier X 城市
- **不投放** paid traffic（Meta / Google / TikTok geo-target 必须排除这些 zip）
- 不主动 list 在 Amazon 这些 metro 的 sponsored placement
- 客服回应来自 Tier X 的咨询时：透明告知"本产品对 chloramine 去除率有限，建议关注 V2 产品 / 其他 catalytic carbon 方案"
