# Bathtub Filter — SNS / Creator / 视觉话语 taxonomy

## 为什么有这页

Gap doc 表中**引用了 `bathtub-filter-sns-creator-and-visual-taxonomy.md` 但文件不存在**，是一个 broken reference。`visual-merchandising-and-creative-strategy` 只覆盖 visual world（baby-safe / ritual / technical），不覆盖**谁在 SNS 上推、用什么句式、什么内容格式实际跑量**。

这页回答：

> **如果 KES 明天上架，要在 TikTok / Instagram / YouTube 上找哪种创作者、用什么内容格式、模仿/避开谁的话语？**

## 1. 一句话结论

这个类目的 SNS 话语**不是去中心化的**——它由**两个超级节点**主导：

1. **品牌自营账号** @filter.baby / @thefilterbaby（FilterBaby）—— 单账号驱动了"chlorine = 皮肤罪魁祸首"整套话语
2. **lab-test 类评测者** Water Filter Guru —— 单 reviewer 决定了"哪个品牌真的去氯"的可信叙事

**KES 在没有自建创作者账号 + 不被 Water Filter Guru 测试通过之前，整个 SNS 战场是别人的话语。**

## 2. 创作者 / 节点地图

### Tier 1 — 品牌自营 mega-driver

| 账号 | 平台 | 角色 | 关键内容格式 | KES 含义 |
|---|---|---|---|---|
| **@filter.baby** | TikTok | FilterBaby 自营 | **"chlorine drop test"**（DPD 试剂滴入水中变黄/不变） + "tap vs filtered" 前后对比 + skincare hack 框架 | **建立了"chlorine-induced acne / dry skin"的标准证明仪式**。任何后入者要不就模仿这个仪式（风险：被识别为跟风），要不就发明新的可视化（成本：教育市场） |
| **@thefilterbaby** | Instagram | FilterBaby 自营，331K followers | clinical / dermatologist-tested 强权威 + Erewhon / Target / Nordstrom / Ulta 零售背书展示 | 用零售面铺 = 信任 leverage。KES 进 Target / Ulta 之前，IG 创意必须解决"我凭什么信你"问题 |

> **关键发现：FilterBaby 已被 Time's Best Invention 选中两次，并已在 Erewhon / Target / Nordstrom / Ulta 上架。** 在 SNS 战场上，KES 面对的不是普通竞争者，是已经把自营账号 → 零售货架 → 主流媒体闭环跑通的对手。

### Tier 2 — 独立 lab-test reviewer（决定可信度）

| 账号/网站 | 平台 | 角色 | 影响力机制 |
|---|---|---|---|
| **Water Filter Guru** (waterfilterguru.com + YouTube) | YouTube + 自建站 | 实测 + 数据驱动 unsponsored | 把 5 个 bath filter（Santevia / Canopy / Sprite / Crystal Quest / Tubo）放进同一测试 → 输出 "Santevia 100% 去氯，Canopy 在 fast flow 只去 50%" 的硬数字。**这种内容会被 Reddit / Quora / 育儿博主反复引用**，成为"客观真相"层。 |

### Tier 4 — 育儿主流媒体

| 媒体 | 角色 |
|---|---|
| Motherly (mother.ly) | Canopy launch piece，registry / parenting 视角 |

**KES 含义：** 育儿 mainstream 媒体写 bath filter 时，会**自动转向"安全 / 温度 / 滑倒"**这套叙事，而非"过滤效果"。 KES PR 进入这一层时，必须把"过滤"包装成"baby-safe ritual"中的一环，不能单独推销技术。

## 3. 内容格式 taxonomy（按已识别跑量的格式分）

### 格式 A — Chlorine Drop Test（最强转化仪式）

- **格式：** 玻璃杯接水 → 滴 DPD 试剂（或类似）→ 自来水变黄/红 → 过滤水保持无色
- **代表：** @filter.baby 全部内容，Ciara Deanne "the chlorine test that shocked me"
- **为什么强：** 把"看不见的氯"变成**3 秒可视化**，触发"我之前竟然在用这个洗澡"的反差
- **KES 用法：** 必须有自己的 chlorine-test 内容（哪怕只是产品页 GIF）。没有 = 在最强转化点缺席
- **风险：** DPD 测试只能证明"自由氯被去掉"，不能证明"chloramine / 重金属 / 软化"。 KES 若用此格式，**必须严格只声明氯**，否则触发 FTC misleading standard

### 格式 B — Tap vs Filtered Before/After（前后对比）

- **格式：** 同一区域 14–60 天前后对比图（皮肤纹理 / eczema patch / hair frizz）
- **代表：** Ciara Deanne 60 天 toddler eczema、StyleShaker 60-day results、Canopy 自传 customer reviews
- **风险：** 任何"湿疹改善"前后图都会被 FDA/FTC 视为 medical claim implication。 KES 用此格式必须**避开 eczema 词、避开患处特写、改用"feels softer / less tight"主观语言**

### 格式 C — Lab/Comparative Test（决定 SEO + Reddit 引用）

- **格式：** Water Filter Guru 风格——多品牌并排、流速差异、浓度数值、unbiased 框架
- **影响层：** 不直接转化，但**决定 12 个月内所有 Reddit / Quora / 育儿博主的引用素材**
- **KES 含义：** 不能跳过这一层。送测策略 = SNS 战略的一部分

### 格式 D — Mom Routine Reel（高 reach，低单转化）

- **格式：** "晚上 8 点 toddler bath routine" + 产品自然出现
- **代表：** TikTok #momtok 子流，Tubo / Canopy 在 routine reel 中露出
- **风险：** routine reel 不解释"为什么需要这个"，**只对已 aware 的妈妈有效**——会被算法推给 awareness=0 的用户但不转化

### 格式 E — Doctor / Derm Endorsement（合规 ceiling 高）

- **格式：** 护肤医生 / 儿科医生 talking-head 解释 chlorine 影响
- **代表：** Interior Medicine, Skin Type Solutions
- **风险：** "Doctor-recommended" 在 FTC endorsement guides 下要求医生**实际使用 + 真实陈述**。付费写"医生推荐"不可

## 4. 话语 / 句式模式（哪些句式是高频转化锚点）

> 以下短语来自直接捕获的 TikTok 视频标题、Canopy 客户评论、creator blog post 标题。**未经 legal review，不可直接用。**

### 4.1 高转化句式（KES 可借力的结构，不可借词）

| 模式 | 实例 verbatim | 触发什么 | KES 改写候选（仍需 legal） |
|---|---|---|---|
| "I can't believe how much X was in my Y" | "I can't believe how much Chlorine was in my tap water 😳" | 反差 + 个人惊讶 | "我之前根本没想到自来水里有这么多氯" |
| "Y'all I'm not letting Z near my skin" | "You're mistaken if you think I'm letting that and Chlorine onto my skin 🙅🏻‍♀️" | 主权 + 拒绝 | "我决定不再让孩子在没过滤的水里洗澡" |
| "the test that shocked me" | "the Chlorine Test That Shocked Me" | 仪式 + 戏剧 | "我做了那个氯测试之后" |
| "noticed [X-time] later" | "noticed fewer flare-ups once using it consistently" | 时间窗 + 谦虚 | "用了 X 周之后皮肤感觉没那么紧" |
| "peace of mind knowing" | "peace of mind knowing bath time isn't going to compromise their child's skin" | 父母焦虑解压 | "洗澡时不再担心水的事" |
| "blends seamlessly into your space rather than screaming 'baby gear'" | Motherly on Canopy | 设计感 vs 功能感 | （KES 可借力：把"不像母婴产品"作为视觉差异化） |

### 4.2 触发法规风险的句式（必须避开）

| 句式 | 风险类别 | 来源 |
|---|---|---|
| "cleared up my eczema" / "got rid of my eczema" | FDA medical claim | Reddit verbatim 多次 |
| "treats / prevents / heals X" | FDA medical claim | 通用 |
| "safe for babies" 单独使用 | CPSIA labeling 触发 | 通用 |
| "doctor-recommended" 未指明来源 | FTC endorsement | 通用 |
| "removes 99.9% of [list]" 无认证 | NSF claim risk | FilterBaby copy 借鉴前必须慎重 |
| "improves blemish-free skin" | FDA cosmetic-vs-drug 边界 | FilterBaby 已用，但其有 clinical study + cert backing |

## 5. 视觉元素 taxonomy（实际跑出量的视觉模式）

### 5.1 已被验证有效的视觉元素

| 元素 | 出现在 | 触发什么 |
|---|---|---|
| **黄色/橙色滴测对比** (DPD chlorine test) | @filter.baby、Canopy 评测 | 立即可视化"问题存在" |
| **柔和奶油 / 燕麦色配色** | Canopy（Oat / Jade / Lavender SKU） | 与 baby gear 美学差异化 |
| **toddler 浴缸内场景，正面，无脸特写** | Canopy / Tubo 广告 | 触发"美好家庭时光"，避开 child-image legal risk |
| **silicone soft cover** 特写 | Canopy 主图 | 双卖点（filter + safety bumper） |
| **温度变色指示器** | Canopy 顶视图 | 视觉差异化 + 安全暗示 |
| **真人手安装 demo（10 秒内）** | YouTube installs | 解除 "tenant-friendly?" 怀疑 |
| **"三重功能" 拼图视觉** | Canopy / Tubo | premium price 的 visual justification |

### 5.2 KES 视觉差异化机会

基于 brand-operating-matrix-v2 + 当前 visual landscape，KES 视觉策略候选：

1. **"工程精度 + ritual 美学" 混合** —— 当前 premium DTC 全偏 ritual / wellness（Canopy / Santevia），技术派偏工业感（Sprite / Crystal Quest）。**没有品牌占据"精密小家电美学"位置**（参考 Dyson / Shark / Anker 美学）
2. **流量可视化** —— 没有品牌真正解决"我洗澡时这个东西在工作吗？"的视觉证据。如果 KES 能做 inline 流量/颜色指示，是新视觉锚点
3. **"safe-for-aging-skin" 反向定位** —— 当前所有视觉都瞄准 baby/toddler，**没人占据 60+ 自照护人群**（claim 风险更低，购买力更高）

## 6. KES SNS 行动建议（按 leverage 排序）

### 必做（V1 launch 前）

1. **送测 Water Filter Guru + mindbodygreen + Interior Medicine** 各 2–3 件样品
   - 时机：定型样品（DV sample）出来后立刻
   - 风险：可能拿到差评 → 必须接受这个风险，否则不存在 = 不被讨论
   - 备份：Onlyzone media-level cert 提前 brief 给 reviewer，降低"未认证"打分扣点

2. **建立自营 TikTok + IG**，至少 3 个月内容池
   - 内容支柱：(a) chlorine drop test demo（每周 1）、(b) install / fit demo（每两周 1）、(c) filter cartridge 替换 demo（每月 1）
   - **不做** mom routine vlog（reach 浪费 + 创作成本高）
   - **不做** before/after eczema 内容（FDA 风险）

3. **Product seeding 给 5–10 个 mom + skincare nano/micro 创作者**
   - 目标 follower 5K–50K（micro 转化高于 mega）
   - 周期 60 天，要求 1 reel + 1 carousel + 1 story set
   - 严禁 brief 中出现"湿疹"、"治愈"、"医疗"措辞

### 可选（A/B 测）

4. 测 "60+ self-care" 反向定位创作者（与 mom 路线 split test）
5. 测 "engineering aesthetic" 视觉 vs "ritual aesthetic" 视觉的 CTR

### 避免

- ❌ 不模仿 @filter.baby 的句式（被识别为跟风 + 风险传染）
- ❌ 不付费"doctor-recommended" 内容
- ❌ 不在 SNS copy 中使用 eczema / cure / treat / heal / safe-for-baby 单独词
