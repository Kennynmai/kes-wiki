## Evidence Boundary

This is a positive first-party fit record for RV / mobile-home center-set faucet samples with central pull-up diverter knobs. It should update the spout-fit matrix as two validated S-02-adjacent sample groups:

- Mobile Home RV Tub Shower Center-Set Faucet
- RV Tub & Shower Faucet Valve Diverter

Do not generalize this into:

- all RV / mobile-home tub faucets are supported
- all center-set faucets are supported
- all S-02 diverter spouts are fully validated
