## L0 - 品牌识别与定位

| 维度 | 内容 |
|------|------|
| 品牌名 | Qure Skincare |
| 官网 | qureskincare.com |
| 主打 Slogan | "Science-Backed Skincare, Clinically-Proven Results" / "Good skin shouldn't be a luxury." |
| 核心定位 | 把"临床级护肤科技"带回家 —— 让专业护肤效果在家就能实现，无需诊所价格 |
| 价格带 | LED Mask $299（预估）、Micro-Infusion $79、Faucet Filter 价格未公开 |
| 目标人群 | 关注临床效果、信任科学背书、愿意为"专业级"付费的护肤用户 |

**定位分析**：  
Qure 与 FilterBaby 的定位最接近——都是**"临床/专业护肤 → 水过滤"** 的扩展路径。差异在于：Qure 从护肤设备（LED 面具、微针）起家，FilterBaby 从水过滤起家。Qure 的"临床背书"来自皮肤科医生委员会，FilterBaby 来自中医背景 + 临床测试。

## L2 - 创始人叙事与品牌故事

### 创始人故事（核心品牌资产）

**Matt Orlic（兄）+ Kristina（妹）双人创始**

> "That belief sparked Qure — a brand built to bridge the gap between clinical-grade treatments and everyday skincare, without the clinic price tag."

**Matt 的个人挫折与转折**：
- 在澳大利亚为大众零售商开发了 25+ 品牌
- **一夜损失 $500 万**：主要零售商 Dick Smith（澳大利亚十亿美元级零售商）破产，Matt 因过度集中（concentration risk）几乎失去一切
- 被迫转型 DTC，学习数字营销，创立数字代理机构帮助电商扩容
- **用代理机构的经验反哺 Qure**："All the lessons we learned in the past and all the skill sets we developed."

**Kristina 的护肤挣扎**：
- 长期以来与皮肤问题斗争，花费多年研究产品和护肤流程
- 她给 Matt 寄"关怀包裹"（带手写说明）→ Matt 的护肤效果来自 LED 面板，而非妹妹的建议
- **核心洞察**：专业科技有效，但价格不亲民、获取不便捷 → 创立 Qure 解决这个问题

## L12 - 机会点

### 对 KES 的机会洞察

1. **Qure 的 Faucet Filter 是"品类扩展"，而非核心**：
   - Qure 主攻 LED 面具/微针，水龙头过滤器是"顺势扩展"
   - FilterBaby/KES 的主攻就是水过滤 → 专注度更高，教育内容可以更深度

2. **Qure 的"临床"是西方医学**，FilterBaby 的"临床"是中医：
   - 两个"临床"叙事可以形成互补（而非直接竞争）→ KES 可考虑"中西医结合"叙事

3. **Qure 未强调"文化独特性"**：
   - 中医/亚洲护肤理念是 FilterBaby/KES 的独特文化资产 → Qure 无法复制

4. **Qure 的创业叙事更适合英语媒体**，FilterBaby 的中医叙事更适合全球社媒：
   - KES 可同时利用：中文媒体讲中医故事、英语媒体讲"临床测试 + 全球护肤新标准"

## L16 - 研究结论

Qure Skincare 是一个**"临床级护肤科技 + 皮肤科医生委员会 + 创业挫折叙事"** 的品牌，与 FilterBaby 的**"中医专业背景 + 临床测试 + 社媒教育"** 形成最有参考价值的对比。
