# 浴缸过滤器类别边界

## 相关页面
- [[bathtub-filter]]
- [[bathtub-filter-product-forms]]
- [[bathtub-filter-user-segments]]
- [[bathtub-filter-research-map]]

## 为什么有这页
`bathtub filter` 不是一个边界天然清楚的标准品类。

这页的作用是把三个问题说清楚：
- 它到底是什么
- 它和哪些邻近类别有关
- 它不应该被误写成什么

## 工作定义
浴缸过滤器是一种用于**浴缸注水场景**的水处理配件，目的是减少特定水质问题，或改善沐浴舒适度。

它通常具有以下特征：
- 面向浴缸注水，而不是淋浴或饮用水场景
- 通过临时或半临时附件工作，而不是改造永久管路
- 用户关注点集中在氯气味、沐浴体感、敏感肌场景、安装便利性等问题
- 商业表达通常强于工程分类，因此边界容易被营销语言拉宽

## 邻近类别
### 淋浴过滤器
关系：
- 最接近的邻近类别
- 市场更成熟、认知更稳定
- 有时会被拿来作为“给浴缸注水”的替代方案

意义：
- bathtub filter 在很多时候是从 shower filter 逻辑延伸出来的
- 但两者并不等同，因为浴缸场景更依赖注水流速、兼容性和溢水控制

### 浴球过滤器
关系：
- 这是 bathtub filter 内部最常见的一种物理形态
- 在消费者语言里，bath ball 有时几乎被当成 bathtub filter 的代名词

意义：
- 不是所有 bathtub filter 都是 bath ball
- 但 bath ball 的确是这个类目里最有识别度的一种形式

### 全屋过滤
关系：
- 常被认为是更完整、工程上更扎实的解决方案
- bathtub filter 则更像局部场景下的轻量替代

意义：
- bathtub filter 本质上是折中方案，而不是终极水处理方案
- 它卖的是进入门槛低、安装轻、场景明确，而不是系统级完整性

### 水龙头 / 台下 / 净水系统
关系：
- 在安装附件或滤材叙事上有部分重叠
- 但目标场景完全不同

意义：
- 这些系统面向饮用、洗漱或厨房使用
- bathtub filter 的重点是沐浴体验和特定敏感场景，不应混写成通用净水设备

## 这个类别不是什么
- 不是永久性的管道过滤系统
- 不是标准意义上的真正软水器
- 不是经过医学验证的湿疹解决方案
- 不是所有“泡澡添加物”或“bath treatment”都能算作 bathtub filter
- 不是可以无条件替代 whole-house filtration 的方案

## 当前边界判断
目前最稳的理解是：
- 类别已存在
- 但市场语言边界模糊
- 这个类别更多是由用户问题驱动，而不是由工程标准驱动

消费者通常不是通过技术结构来理解它，而是通过下面这些问题：
- 这会不会让宝宝或敏感肌洗澡更安心？
- 这会减少氯味吗？
- 这对敏感肌肤有帮助吗？
- 我能不能在出租屋或公寓里轻松装上？

## 为什么这对 KES 重要
如果 KES 要进入这个类别，不能假装它比现在更标准化、更技术闭合。

更合理的做法是先决定，自己要站在哪个子边界里：
- 婴儿 / 敏感肌沐浴配件
- 去氯 / 沐浴舒适度配件
- 租房友好的轻安装方案
- 高端沐浴升级产品

边界一旦选错，就会同时带来错误的宣称、错误的适配承诺和错误的用户预期。
