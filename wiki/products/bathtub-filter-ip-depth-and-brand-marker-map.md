# Bathtub Filter 竞争 IP / 品牌标记深度地图

## 为什么有这页
这页不做法律意义上的 FTO，
而是回答更实用的问题：

> **哪些品牌看起来更像有真实技术 / 设计锚点，哪些更像主要靠 marketing posture（营销姿势）在卖？**

## 当前一眼判断
### 1. 这个类目的 IP 深度并不均匀
目前更像分成三类：

#### A. media / chemistry lineage 比较清楚
代表：**Sprite adjacency**
- hot-water dechlorination
- calcium sulfite / copper-zinc 邻接
- shower-lineage chlorine media story

这类更像 **real technical heritage（真实技术血统）**。

#### B. form-factor / configuration lineage 清楚，但 moat 不一定来自强功效证明
代表：**Crystal Quest bath-ball route**
- bath-ball 形态长期存在
- 公开 surface 现在还能看到 design-patent 线索（如 Bath ball filter design patent `D1034899`）

这类更像 **design / form continuity（设计与形态延续）**，
不等于它天然拥有最强 efficacy moat。

#### C. brand-layer conversion 强，但 public patent anchoring 弱
代表：**Canopy / Santevia / Kinder / Tubo**
- route identity 很清晰
- 文案 / 设计 / segment 很强
- 但当前可见前台并没有像 Sprite 那样高频展示明确 patent / media lineage

这类更像 **brand system moat（品牌系统护城河）**，而不是明显的 public-facing utility-patent moat。

## 2. 这个类目里要区分 3 种“护城河幻觉”
### Utility-patent-adjacent
通常对应：
- media chemistry
- hot-water chlorine reduction logic
- shower/tub cartridge architecture
- forced-flow / spout-filter configuration

这类更支持 **技术可信度**。

### Design-patent-adjacent
通常对应：
- bath-ball shell
- over-faucet shell form
- distinct exterior silhouette

这类更支持 **外观区分度**，
不自动支持 filtration claim credibility。

### Mostly marketing-language-led
典型表现：
- clinically tested
- patent pending
- 8-stage / 17-stage
- premium baby / eczema / safe-water theater

这类更支持 **转化氛围**，
不代表真正的技术壁垒。

## 3. 品牌逐个看
### Sprite
当前最像：
- **technical-lineage-led**

原因：
- 与 calcium sulfite / hot-water dechlorination patent lineage 邻接最强
- shower-lineage 与 narrower chlorine story 更一致

结论：
- 是当前最值得 KES 参考的“技术收敛型样本”

### Crystal Quest
当前最像：
- **form-factor + cartridge-configuration route**

原因：
- bath-ball route 长期可见
- 公开 surface 已出现 design-patent 线索
- 但并没有同步看到同等级别的窄功效护城河表达

结论：
- 比“只是营销”更强一点
- 但强项更像外形 / route continuity，不是最干净的 proof moat

### Canopy
当前最像：
- **design-first / segment-first moat**

原因：
- baby registry / wellness / premium object 逻辑非常清楚
- bath SKU 前台仍以设计、颜色、spout-cover、registry、gentle bath world 为主

结论：
- 它的 moat 更像品牌系统和 visual discipline
- 不是强 public patent signaling 样本

### Santevia
当前最像：
- **ritual / wellness object route**

原因：
- mineral / gentle nourishment / bath ritual story 明显
- hanging accessory 逻辑比 patent-anchor 更突出

结论：
- 更偏 wellness-led，不是 obvious patent-led

### Kinder
当前最像：
- **clinical-positioning DTC without strong visible IP anchor**

结论：
- 转化语言强
- 可见 IP 深度弱
- 更像用 positioning 去替代 moat impression

### Tubo
当前最像：
- **performance-claim-heavy DTC / marketplace hybrid**

结论：
- claim energy 很强
- 可见技术 / patent 锚点仍弱
- 更像 marketing-led growth posture

## 4. 当前更重要的不是“谁有专利”，而是“谁在借什么东西建立可信感”
目前看，可信感主要来自三种来源：
1. **技术 lineage** — Sprite 最典型
2. **设计 / visual distinctiveness** — Canopy、部分 Crystal Quest
3. **clinical / parenting / testimonial theater** — Kinder、Tubo 等更明显

这三种来源对 KES 的意义完全不同。

## 5. 对 KES 的直接启发
### 不要误判
- 高转化 ≠ 高 IP 深度
- 有 design patent ≠ 有强性能护城河
- 没快速看到 patent ≠ 一定没有保护

### 更值得学的
- Sprite 的窄技术血统
- Crystal Quest 对 form factor continuity 的启发
- Canopy 的 design-system / segment packaging

### 更不该照抄的
- 用 clinically tested / baby-safe / eczema-forward 当作 moat 替身
- 用 patent-pending 氛围替代明确 route discipline

## 当前一句话结论
**bathtub filter 赛道里，真正像“技术护城河”的公开样本仍偏少；更多品牌是在卖 design moat、segment moat，或 pure conversion theater。**
