# 埃及租厂房落地检查清单

## Trigger / scope
- 适用于以下场景：
  - 香港主体控股埃及公司；
  - 原材料和设备主要从中国进口；
  - 成品主要内销埃及；
  - 初期采用 `租厂房` 而不是买地自建。
- 目标不是一次性覆盖所有法律细节，而是防止在签租约、买设备、招人和报关前踩进不可逆错误。

## Inputs
- 拟售产品清单：成品窗帘、布帘、轨道系统、辅料、安装服务。
- 候选厂房清单：地址、面积、租金、租期、房东主体、现有许可状态。
- 初步设备清单：裁剪、缝制、压褶、定型、包装、仓储设备。
- 初步原材料清单：布料、遮光层、辅料、轨道、包装。
- 目标客户清单：家装、酒店、地产、商装、工程渠道。
- 埃及本地顾问名单：律师、税务师、报关行、会计师、劳工/牌照顾问。

## Steps

### 1. 先做业务模式闸门
- 确认你们做的是哪一种：
  - 整品从中国进口直销埃及；
  - 中国来料，埃及做轻加工/组装；
  - 中国来料，埃及做完整本地加工与安装。
- 必做一版单位经济对比：
  - `整品进口直销`
  - `来料加工内销`
- 只有在后者至少在毛利、交期、周转、工程中标率里明显胜出时，才进入租厂房和上设备阶段。

### 2. 锁定法律结构
- 默认 base case：`香港控股公司 -> 埃及内陆 LLC -> 埃及本地销售/安装/生产`
- 让埃及律师确认：
  - LLC 是否足够覆盖进口、生产、分销、安装；
  - 是否需要额外商业登记、工业登记或行业许可；
  - 房东名下厂房当前用途与承租后拟从事活动是否匹配；
  - 外资股东、经理、签字权和董事会安排。
- 在没有强理由前，不把自由区作为默认选项。

### 3. 先审厂房合法性，再谈价格
- 向房东索取并核验：
  - 产权证明或可出租权证明；
  - 厂房平面图和面积；
  - 现有工业许可或运营许可；
  - 消防验收或相关合规文件；
  - 电力容量和历史公用事业账单；
  - 是否存在抵押、查封、诉讼或政府争议。
- 确认房东有权把该厂房租给一个外资控股制造公司，而不是只能做仓储或商贸。

### 4. 确认活动与许可匹配
- 用 IDA 的工业活动查询去确认拟从事活动的分类和许可方式。
- 不要只问“能不能做纺织”，要具体到：
  - 窗帘缝制；
  - 布料裁剪；
  - 配件组装；
  - 包装；
  - 仓储；
  - 安装服务。
- 如果厂房现有许可不匹配，先问清：
  - 是可变更、可补证，还是要换厂房；
  - 由房东负责还是由承租方负责；
  - 补证期间能否合法开工。

### 5. 评估厂房技术适配
- 实地检查：
  - 楼层承重是否适合设备和仓储；
  - 电力容量是否足够，是否需要增容；
  - 进出货动线是否顺畅；
  - 是否有足够布料仓和成品仓；
  - 排风、除尘、空压、照明、消防喷淋是否到位；
  - 装卸货口、货车进出、停车和安保条件。
- 对每一台核心设备标记：
  - 额定功率；
  - 尺寸；
  - 对地基/地坪要求；
  - 是否需要吊装或特殊安装。

### 6. 算清进口链条税负与现金流
- 对设备、布料、辅料、轨道、包装材料逐项建立表格：
  - 产品名称；
  - 供应商；
  - HS code；
  - 中国出口口径；
  - 埃及进口关税；
  - 增值税；
  - 其他口岸费用；
  - 清关周期；
  - 最小起订量；
  - 安全库存天数。
- 不允许只用一个“平均税率”做模型。
- 单独测算首批开工所需的营运资金峰值：
  - 海运在途；
  - 清关占用；
  - 布料库存；
  - 在制品；
  - 应收账款。

### 7. 审租约，不只看租金
- 租约必须明确：
  - 租期；
  - 续租权；
  - 提前解约条款；
  - 押金；
  - 租金递增机制；
  - 免租装修期；
  - 装修和机电改造谁承担；
  - 消防整改谁承担；
  - 设备安装许可；
  - 设备搬离权；
  - 转租/分租限制；
  - 房东违约救济。
- 如果租约不允许你们安装重设备、改排风、改电力、改仓储动线，这个厂房基本就不适合。

### 8. 先做最小可运行工厂
- 第一阶段优先配置：
  - 最少必要设备；
  - 小团队；
  - 能交付的核心 SKU；
  - 简化仓储和质检流程。
- 不要第一天就按“大工厂”建。
- 目标是先验证：
  - 交期；
  - 良率；
  - 工程单响应速度；
  - 安装能力；
  - 回款纪律。

### 9. 建立本地交付能力
- 如果产品以内销埃及为主，必须尽早确认：
  - 本地测量团队；
  - 安装团队；
  - 售后返工机制；
  - 项目管理和现场协调；
  - 阿拉伯语客户沟通能力。
- 对窗帘这种品类，很多价值不在制造本身，而在 `按单定制 + 现场交付 + 返工响应`。

### 10. 设立开工前的红线
- 以下任一项未确认，不开工：
  - 厂房出租权不清晰；
  - 工业活动与许可不匹配；
  - 电力容量不满足；
  - 关键设备无法合法安装；
  - 租约不保障设备搬离；
  - HS code 和税负模型未做完；
  - 单位经济尚未证明本地加工优于整品进口；
  - 本地经理或安装团队未到位。

## Failure modes
- 把“租到厂房”误当成“可以合法生产”。
- 先签长租，再发现活动许可不匹配。
- 低估进口税费、清关时间和营运资金占用。
- 只看低工资，忽略安装、返工和渠道成本。
- 为了节省首期租金，选了不能增容、不能改造、不能走货的厂房。
- 设备已到港，厂房消防、电力、排风仍未办妥。
- 把香港控股误当成自动税务优化工具。

## Review notes
- 这份清单更偏投前和开工前控制，不代替埃及本地法律意见。
- 后续如果确定具体城市或工业区，应再分一版区域化清单。
- 如果后续改成买地建厂，这份协议和许可检查逻辑仍有用，但土地、建设和验收模块要重写。
