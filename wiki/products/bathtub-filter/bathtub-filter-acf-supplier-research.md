## 一、概述

### 1.1 背景
KES 浴缸过滤器产品需要高效滤材以去除消毒副产物（THMs、HAAs）。ACF（活性炭纤维）相比传统颗粒活性炭（GAC）和烧结碳块，具有：
- **极高流速**（压降 <0.05 bar @ 20L/min）
- **零掉粉风险**（纤维毡形态，无颗粒脱落）
- **快速吸附**（接触时间 0.5–2 秒即可达 80%+ THMs 去除率）
- **薄型设计**（5–10mm 厚度即可，滤芯可做得非常紧凑）

### 1.2 ACF 基材选择
| 基材 | 特点 | 推荐度 |
|------|------|--------|
| **PAN 基**（聚丙烯腈） | 性价比好，微孔发达，吸附容量高 | ⭐⭐⭐⭐⭐ 首选 |
| 沥青基 | 吸附容量最高，成本较高 | ⭐⭐⭐ 高端应用 |
| 粘胶基 | 成本低，但性能较弱 | ⭐⭐ 不推荐 |

**结论：** KES 产品应选用 **PAN 基 ACF 毡**。

## 三、产品规格参考（典型 PAN-ACF 毡参数）

### 3.1 关键技术参数

| 参数 | 典型值 | KES 浴缸过滤器建议 |
|------|--------|---------------------|
| **BET 比表面积** | 900–1600 m²/g | **>1200 m²/g** |
| **厚度** | 2–10 mm | **5–8 mm**（折叠使用） |
| **吸附 THMs 容量** | 20–40% wt | — |
| **压降**（20L/min·dm²） | <0.05 bar | ✅ 优秀 |
| **基材** | PAN（推荐）/沥青/粘胶 | **PAN 基** |
| **使用温度** | <150°C（水用无限制） | — |
| **再生性** | 可热再生（~120°C） | 家用滤芯不再生 |
| **载银处理** | 可选 | ✅ 推荐（抑菌） |

### 3.2 折叠式 ACF 设计参数

```
折叠式 ACF 设计：
- 有效截面积：φ80mm ≈ 50 cm²
- 折叠后有效面积：150–250 cm²（3–5× 增加）
- 在 20L/min 流量下，实际面流速：0.8–1.3 L/min·dm²
- THMs 去除率：80–88%
```

## 五、采购建议

### 5.1 第一步：先拿样品测试
向以下两家供应商各申请 ACF 毡样品：
1. **江苏苏通碳纤维有限公司**（首选）
2. **鞍山活性炭纤维厂**（次选）

**样品要求：**
- 基材：PAN 基
- 厚度：5–8 mm
- BET 比表面积：>1200 m²/g
- 是否载银：是（抑菌）
- 样品尺寸：A4 大小（210×297mm）或 φ80mm 圆片

**测试指标：**
- THMs 去除率（目标 >80%）
- 压降（目标 <0.05 bar @ 20L/min）
- 掉粉情况（目测 + 浊度测试）
- 机械强度（是否易碎/掉纤维）

### 5.2 第二步：询价重点问题

向供应商询价时，需明确以下问题：

1. **最小起订量（MOQ）**
   - ACF 毡的 MOQ 是多少平方米？
   - 是否接受小批量试样订单（<100 m²）？

2. **价格分层**
   - 厚度 5mm / 8mm 的单价
   - 按 100m²、500m²、1000m² 分别询价
   - 是否可提供已裁切尺寸（如 φ80mm 圆片）？

3. **加工服务**
   - 是否可提供复合加工（ACF + PP 骨架支撑）？
   - 是否可做折叠式滤芯成品？
   - 是否可提供载银处理？

4. **技术文档**
   - 是否可提供完整技术数据表（TDS）？
   - 是否可提供 NSF/ANSI 认证相关测试报告？
   - 是否可提供 SGS 检测报告？

5. **交期和物流**
   - 批量订单（500–1000 m²）的交货周期？
   - 是否支持快递发货到香港/江门？
   - 是否提供出口报关服务（如需出口到美国）？

### 5.3 第三步：考虑成品滤芯供应商

如果自行加工 ACF 有难度（需要折叠、封装、加支撑骨架），可直接联系：

**厦门米兹净水科技有限公司**（www.mizitec.com）
- 专做 ACF 复合滤芯成品
- 湿法成型技术，亲水性能优异
- 可直接供应成品滤芯，降低 KES 加工难度
- **建议：** 同时向米兹净水询价成品滤芯，与自行加工做成本对比

## 六、需要进一步确认的信息

由于网络抓取限制，以下信息建议直接联系供应商确认：

1. **精确技术数据表（TDS）**
   - BET 比表面积测试方法（N₂吸附）
   - 孔径分布（微孔/中孔/大孔比例）
   - 吸附等温线（THMs、氯仿）
   - 压降曲线（不同流速下的压降）

2. **NSF/ANSI 认证情况**
   - 进入美国市场需要通过 NSF/ANSI 42 和 53 认证
   - 询问供应商是否已有相关认证或可协助测试

3. **样品政策**
   - 是否提供免费样品？
   - 样品费用是否可抵扣批量订单？

4. **质量控制**
   - 出厂检验标准（首次冲洗出水浊度、碳粉脱落量等）
   - 是否可提供每批次的检测报告？

5. **定制服务**
   - 是否可根据 KES 的滤芯设计定制 ACF 毡的尺寸和形状？
   - 是否可在 ACF 毡中添加其他功能材料（如 KDF-55 粉末）？
