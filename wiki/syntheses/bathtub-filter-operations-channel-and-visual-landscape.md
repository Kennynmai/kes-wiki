## 为什么会有这页
这页覆盖的是当一个品类被证明值得进一步关注之后，会变得重要的 second-tier 与 third-tier layers（第二层、第三层研究层）。

这些层关注的重点，不再是“这个担忧是否真实”，而是“这个品类在运营与商业上是否真的跑得通”。

## 安装 / 兼容性的现实
目前已浮现的信号指向一个很实际的事实：
- bathtub filters 并不处在一个标准化安装环境里
- faucet shape、spout geometry、attachment method、fill speed 都很关键
- 有些 workaround routes（替代路径）本质上是把问题变成 shower-to-bath filling setup，而不是原生 tub-spout solution

### 早期运营风险信号
- 市场上的 universal-fit 语言可能被夸大了
- flimsy plastics（脆弱塑料件）或 attachment stability 差，会很快摧毁信任
- 如果想让 chlorine reduction 仍然有意义，某些架构下 flow-rate reduction 可能几乎不可避免

## 定价 / 补充装逻辑
目前浮现出一个常见模式：
- 2,500 gallons 经常被作为可见 cartridge life benchmark（滤芯寿命基准）
- 不同品牌与 listings 的 replacement timing claims 差异很大
- 有些品牌强调 long-life，但当前表面 claims 并不一致，因此需要 usage-model sanity checks（使用模型合理性校验）

### 为什么重要
如果一个 bathtub filter 一边明显降低 fill rate（注水速度），另一边又需要比较频繁地更换，那么它在真实使用中的体验，很可能会比 conversion copy（转化文案）看上去更差。

## 渠道版图
目前看，这个品类至少横跨四种 channel identities（渠道身份）：

### 1. Marketplace / commodity
- Amazon
- Walmart

典型叙事：
- broad claims
- replacement bundles
- easy-install positioning
- 激烈竞争与 generic lookalikes

### 2. Utility / home-improvement
- Home Depot 风格渠道存在

典型叙事：
- 更靠近 hardware-adjacent trust（五金邻近型信任）
- 更强调安装、兼容性与 replacement parts

### 3. DTC / premium wellness
- Santevia、Canopy、Tubo、Kinder 一类路径

典型叙事：
- skin comfort
- baby-sensitive-skin framing
- 更柔和的设计语言
- 更强的情绪转化

### 4. Reseller / educator / community layer
- wellness retailers
- review publishers
- parent / community spaces

典型叙事：
- explanation 与 reassurance
- 常常承担连接 skepticism 与 conversion 的桥梁作用

## 社交 / 创作者层
短视频与 creator content 对理解市场在情绪上觉得什么最有吸引力，似乎特别有用。

### 当前信号
这个品类很容易被压缩成一个简单的 short-form promise（短内容承诺）：
- gentler baths
- less chlorine exposure
- 更适合 babies / sensitive skin / eczema-prone households

### 风险
短内容也很容易把细节抹平，并把这个品类推向 skin outcome framing（皮肤结果导向叙事）等更高风险区域。

## 视觉陈列方向（Visual-merchandising directions）
目前至少已经能看到三种 merchandising modes（陈列模式）：

### A. Baby-safe / gentle-parenting
- 柔和色彩
- child / family reassurance
- 暗示 skin-soothing

### B. Premium bathroom ritual
- 平静、提升感、wellness-oriented 的呈现
- 强调 self-care 与 bath experience

### C. Technical filtration credibility
- 滤材 stages、contaminant lists、filter internals、cartridge language

## 战略含义
这个主题已经明显横跨三层研究：
- 第一层：evidence、reviews、claims / compliance、brand-IP cues
- 第二层：installation、refill economics、channels
- 第三层：social compression、creator discourse、visual-selling logic

这意味着，这个品类现在应该被视为一个 full-stack product-research topic（全栈式产品研究主题），而不只是一个狭义的 evidence scan。
