## 三种更换逻辑语言
### 1. gallons（加仑）语言
典型例子：
- 2,500 gallons
- 4,755 gallons

优点：
- 听起来技术化、可量化
- 适合 listing 与包装展示

问题：
- 普通用户并不会自然把 gallons 换算成 bath count 或 household months
- 很容易形成“寿命很长”的 impression，但不一定反映真实 bath-use burden

### 2. bath-count（沐浴次数）语言
典型例子：
- 100 baths
- 25–30 baths

优点：
- 对 bathtub 场景最直观
- 更容易被用户理解为“我家能用多久”

问题：
- 仍受 tub size、注水量、是否给孩子洗澡、使用频率影响
- 如果没有说明边界，也会留下解释空间

### 3. time（时间）语言
典型例子：
- every 90 days
- 6–8 months
- 10–12 months

优点：
- 最适合 subscription
- 最易于 retention / reminder / CRM 设计

问题：
- 如果没有 usage anchor（使用锚点），其实是最不透明的一种
- 同样 90 天，对低频和高频家庭的真实经济学差很多

## 当前最重要的商业判断
### 1. bathtub filter 的单位经济不只是耗材成本，而是“总负担经济学”
对这个类目，用户实际在承担的不只是 cartridge price，而是：
- 首购价格
- 更换频率
- 注水变慢造成的时间成本
- overflow / slip / awkward setup 风险
- 晾干 / 存放 / hygiene burden
- 对“它到底还有没有效果”的不确定感

所以 bathtub filter 更准确地说，是 **financial economics + behavioral economics + trust economics** 的叠加。

### 2. 纸面上最漂亮的 economics，未必是最好经营的 economics
例如：
- 很长的 months claim
- 很高的 gallon claim
- 很 premium 的定价

这些都能让表面 unit economics 变漂亮，
但如果同时伴随：
- slow fill
- fit variance
- chlorine benefit skepticism
- unclear replacement boundary

那 lived economics（真实使用经济学）就会恶化。

### 3. 高价 SKU 需要更强的“证明闭环”
当价格进入 ~$79–$89 甚至更高时，
用户不会只问“能不能装”，还会问：
- 为什么这么贵？
- 正常水流下到底有没有用？
- 90 天为什么要换？
- 如果我只是偶尔泡澡，是不是不划算？

这就是 premium bathtub filter 的核心矛盾：
**毛利空间越好看，证明负担也越高。**

## 对 KES 的实际启发
### 更好的 replacement communication 方式
如果 KES 继续推进，可能不应只写：
- 2,500 gallons
- every 90 days

更合理的是同时给出：
- bath-count range（预估可支持多少次 bath）
- household usage assumption（按什么频率估）
- recommended replacement boundary（什么情况下应更早换）
- high-pressure / large-tub disclaimer（哪些使用场景会更快消耗）

### 更好的 unit-economics 判断框架
KES 以后判断是否值得做，不该只看 gross margin，
而应至少一起看：
1. 首购价格能否撑住渠道身份
2. refill cadence 是否会引发 maintenance fatigue
3. 正常注水下性能是否足够真实
4. 更换逻辑是否能被用户理解
5. review 里会不会迅速出现“贵、慢、麻烦、没感觉”的组合差评

## 当前阶段的简短结论
### 表面最容易讲故事的 economics
- Crystal Quest / generic gallons-language routes
- Canopy premium subscription route

### 当前更像真实可运营 economics 的路线
- Santevia-style bath-count + shorter, clearer replacement logic
- Sprite-style utility / replacement-part logic

### 当前最该警惕的 economics 陷阱
- 长寿命印象很好看，但没有 realistic usage boundary
- premium price 很诱人，但没有 strong proof loop
- 只用 gallons / months 讲话，不把 bath-use reality 讲清楚
