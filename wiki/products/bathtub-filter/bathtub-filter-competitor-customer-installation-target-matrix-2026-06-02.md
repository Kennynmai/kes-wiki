# 竞品客户实际安装对象矩阵（2026-06-02）

## 口径

这里回答的是：客户实际会把每个竞品装在什么东西上。

注意几个词的边界：

- **tub spout / bath spout**：浴缸墙面伸出的出水嘴，是最核心安装对象。
- **tub faucet / bath faucet**：Amazon 和评论里经常泛称浴缸出水龙头，很多时候实际就是 tub spout。
- **widespread faucet / deck-mounted faucet**：双把/三孔/宽距或台面式浴缸龙头。本批没有足够证据证明任何竞品稳定支持这类安装。
- **with overflow**：很多低价产品标题里的 `with overflow` 更像“防溢流/防飞溅结构”，不是安装在浴缸 overflow drain 口。

**2026-06-03 口径修正：** Filterbaby `B0FNVDJRSQ` 是 shower filter / showerhead inline 产品，已拆到 shower-filter 项目。下表只保留 active bathtub-filter 竞品；Filterbaby 的安装对象为 shower arm / wall pipe / showerhead / hose，不再计入 bathtub fit 判断。

## wide / widespread faucet 判断

| 类型 | 本批证据 | 能不能写成支持 |
|---|---|---|
| wide faucet | Santevia 有用户说“wider faucet”需要剪 zip tie 才 fit；SHLLKTTRY 有一条 `wide faucet fit` 标题 | 只能写“有个别 workaround / anecdote”，不能写成支持 |
| widespread faucet / deck-mounted bathtub faucet | 本批未看到明确图证或稳定评论证据 | 不能写支持 |
| regular / standard tub faucet | Beati、JYFJYF、Canopy、Crystal Quest、SHLLKTTRY 等都有 PDP 或评论语言 | 只能写“面向 standard tub faucet/spout”，不能写 universal |
| no-diverter / old faucet | Tubo 评论明确出现无 diverter knob 不稳；Syvahome 也有 diverter 依赖线索 | 必须作为风险场景 |

## 对 KES 的直接含义

1. **KES 不应该把 tub faucet 和 widespread faucet 混写。** 竞品的大多数“faucet”实际是 tub spout，不是三孔/宽距/台面式 faucet。
2. **V1 文案应写 supported tub spout types，而不是 `fits all tubs`。**
3. **最值得对标的是 Canopy 的 tub-spout-cover route 和 JYFJYF/Syvahome 的 strap workaround。**
4. **widespread/deck-mounted faucet 应默认列为未验证或不支持，除非未来有实物/图片证据。**
5. **带 diverter knob 的 straight tub spout 是高优先级测试对象。** Tubo/Syvahome 的评论已经提示 diverter/stopper/knob 会影响固定。
