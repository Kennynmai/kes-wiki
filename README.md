# KES Company Wiki

KES 的共享业务知识系统与组织记忆层。

## Purpose
- 支持平台、市场、产品、项目、决策、SOP、风险与指标管理
- 让重要分析和问答沉淀为可复用知识
- 面向未来全员可读、分层维护、Agent 协同治理

## Core flow
capture → classify → narrow → extract → deepen → update → verify → lint

## Key directories
- `inbox/`：输入缓冲层
- `raw/`：原始证据层
- `wiki/`：维护后的正式/工作知识层
- `governance/`：治理规则
- `ops/`：队列、审查与流程状态
- `templates/`：标准模板
- `dashboards/`：总览、onboarding、official surface

## First-use rule
默认先看：
1. `index.md`
2. `dashboards/official-start-here.md`
3. `dashboards/onboarding-start-here.md`
4. `governance/context-budget.md`
5. `AGENTS.md`
