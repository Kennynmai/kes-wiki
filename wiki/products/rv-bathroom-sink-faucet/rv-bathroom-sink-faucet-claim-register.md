# RV Bathroom Sink Faucet Claim Register

## Why this page exists
This page prevents product copy from outrunning evidence.

Every meaningful claim should be classified as:
- allowed now
- conditional pending evidence
- banned / avoid

## Allowed Claims If Product Matches Spec
| Claim | Evidence needed | Notes |
|---|---|---|
| `4-inch centerset` | Dimension drawing | Must match actual hole spacing |
| `2-hole installation` | Product design + install guide | Include hole-size boundaries |
| `RV bathroom sink faucet` | Product use case and dimensions | Avoid if packaging is actually residential-only |
| `direct replacement for many RV bathroom sinks` | Fit matrix | Use `many`, not universal |
| `ceramic cartridges` | BOM / supplier spec | Must be exact |
| `metal threaded connections` | Teardown / BOM | Specify if only threaded sections are metal |
| `extended shanks` | Dimension drawing | Include length |
| `measured spout reach` | Dimension drawing | Useful against short-reach complaints |
| `leak-tested before packing` | QA process | Must be actually implemented |

## Conditional Claims
| Claim | Required evidence card | Risk if unsupported |
|---|---|---|
| `lead-free` | SKU-level NSF 372 / equivalent evidence or legally reviewed support | Legal / compliance risk |
| `NSF/ANSI/CAN 61` | SKU listing or certificate | Cannot borrow from other KES faucet SKUs |
| `cUPC` | IAPMO listing / model mapping | Cannot use based on company-level certificate alone |
| `ASME A112.18.1 / CSA B125.1` | Listing or test evidence | Distinguish tested vs certified |
| `WaterSense` | EPA / IAPMO WaterSense SKU listing | Flow and model must match |
| `CEC compliant` | SKU and flow evidence | State-specific risk |
| `ADA compliant` | Handle / install criteria evidence | Do not copy competitor language without proof |
| `lifetime warranty` | Legal warranty terms | Must be serviceable |

## Avoid / Banned Claims
| Claim | Reason |
|---|---|
| `universal fit` | RV vanities and supply connections vary |
| `never leaks` | Absolute performance claim |
| `lifetime drip-free` | Overbroad unless backed by warranty and testing |
| `all metal` | Only if literally true across body / waterways / handles / connections |
| `meets all US standards` | Too broad |
| `drinking-water safe` | Use specific standards instead |
| `no plumber ever needed` | Some installations may be modified or tight |

## Copy Red Lines
Before launch, review:
- Amazon title
- bullets
- A+ images
- packaging
- insert card
- warranty page
- customer-service macros

No certification language should appear in any of these surfaces without a completed evidence card.
