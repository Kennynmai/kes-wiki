# 来源摘要｜Bathtub Filter Amazon US 10-ASIN 评论与竞品项目调查（2026-04-22）

## Why this source matters
这份来源把 bathtub-filter 集群此前偏侦察式的 review mining，补成了一个更接近项目决策层的综合包：
- 10 个 Amazon US ASIN
- 2,562 条评论汇总
- 页面卖点、评论反馈、竞品状态三层交叉
- 可直接转成竞品拆解优先级、P0/P1/P2 动作建议

## Extracted facts / observations

### 1. 需求核心不是“去氯参数”，而是“安心 + 省心”
页面卖的是去氯；用户买的是安心。用户最终验证的是：
- 稳不稳
- 省不省心
- 是否适合宝宝 / 敏感肌 / 家庭场景

### 2. 需求主轴有三条
按评论聚类，项目级最高频需求为：
- 硬水 / 氯味 / 水质改善：1128 次，44.0%
- 安装方便 / 不想折腾：660 次，25.8%
- 敏感肌 / eczema / 干痒：592 次，23.1%
- 宝宝 / 儿童洗澡场景：585 次，22.8%

这说明类目的真实购买逻辑是：
- 功能线：硬水 / 氯味改善
- 体验线：安装省事、不折腾
- 场景线：宝宝 / 敏感肌更安心

### 3. 结构与维护比滤材级数更能决定满意度
项目级痛点排序为：
- 漏水 / 溢水 / 水流异常：284 次，11.1%
- 寿命短 / 更换成本 / 维护麻烦：233 次，9.1%
- 说明不清 / 使用边界不清：50 次，2.0%
- 不兼容 / 装不上 / 固定不稳：48 次，1.9%
- 做工 / 结构不稳：41 次，1.6%
- 过滤效果不明显：11 次，0.4%

含义：
- 用户先验证结构表现，再判断过滤效果
- 维护与换芯是第二条真实流失路径
- 兼容问题属于低频高杀伤

### 4. 页面与评论存在四个稳定错位
- 页面讲去氯 / 级数，评论问会不会漏 / 会不会慢
- 页面讲 universal fit，评论问到底能不能装稳
- 页面讲多级滤材，评论问长期值不值 / 怎么维护
- 页面讲 baby / sensitive skin，评论问体感是否真实、是否更安心

### 5. 10 个 ASIN 可以分为三组

- B0742KFY9R

可局部参考：

- B0FL24SLM5

建议避开 / 不建议照搬：

### 6. 更值得推进的路线被进一步收窄
报告推荐两条主方向：
- 中价专业结构款
- 母婴安全方案款

不建议继续做标准化低价白牌挂装过滤款，除非结构稳定性、母婴场景、安全表达、兼容边界或页面证据体系能明显领先。

## Caveats / ambiguity
- 这是项目级综合报告，不是底层 verbatim 评论导出
- 评论统计依赖用户提供的 Excel 汇总，尚未在本仓库内逐条 spot-check
- 某些 SKU 的类型 / 参考价值判断属于策略归纳，不应当作最终 marketplace truth
