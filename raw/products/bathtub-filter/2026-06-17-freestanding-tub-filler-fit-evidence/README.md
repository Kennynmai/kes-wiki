## KES installation setup

- Accessory setup: 附赠的5孔总长124mm，宽20mm的扎带 + 附赠 3M 贴挂钩
- Retention logic: 主固定力来自 5 孔硅胶扎带的拉力和硅胶摩擦力；3M 胶只是辅助防滑层，用来避免硬性 ABS 挂钩与金属浴缸嘴接触面之间滑动，不作为主要承重结构
- Tie-hole position used in the test: 扎带只扎到倒数第二个孔
- Tighter optional position: 扎到倒数第三个孔会更紧，但需要较大拉力组装，对女性用户不太友好
- Required tie position: 扎带需要扎在距离出水嘴末端 60 mm 处

## Result

- Fit result: usable / positive fit evidence
- Load-bearing result: 2 kg static load-bearing tested
- User-observed caution: gravity will pull the tie downward and deform it; if the tie is installed too close to the outlet end, it can slide off

## Evidence boundary

This record supports the following claim:

- Freestanding tub filler with curved tube / special-shaped tube / non-waterfall outlet can be used with the KES accessory setup when the tie is installed about 60 mm from the outlet end.

This record does not automatically support:

- waterfall / wide sheet-flow tub fillers
- wide-body decorative outlets
- dynamic high-flow splash / overflow performance
- long-term fatigue after repeated fill cycles
- all freestanding tub fillers regardless of tube diameter, surface finish, curvature, or outlet geometry
