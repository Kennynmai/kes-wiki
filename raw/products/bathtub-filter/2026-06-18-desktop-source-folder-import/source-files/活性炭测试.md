# 活性炭

# 淋浴常见品种：

椰子壳、果壳（杏仁壳、核桃壳、桃核等）、木质活性炭、煤质活性炭

## 行业尺寸命名：

柱状炭讲直径，颗粒炭讲目数

椰子壳：颗粒状态，所以是目数；

常见尺寸：5-8目、8-10目、6-12目、8-16目；数值越大，颗粒尺寸越小。

吸附速度：颗粒越小越好

过滤精度：颗粒越小越好

水流阻力：颗粒越大越小阻力

使用寿命：颗粒越大越长

# 掉粉程度（灰分低，即掉粉低）和硬度问题、碘值，即过滤效率：

按照我们目前的需求是：灰分低、硬度高

由于我们单个产品的用量低：50g-100g左右，成本变化不大；所以基本上可以确认的是用 “椰子壳活性炭”

ꔷ **椰壳活性炭**：

  **灰分**：一般在2% - 4% 之间。

  **硬度**：具有较高的硬度，一般在95%以上。

  **碘值**：一般在1000 - 1200 mg/g 之间

ꔷ **果壳活性炭**：

  **灰分**：一般在3% - 5% 之间。

  **硬度**：硬度较高，一般在85% - 90% 之间。

  **碘值**：一般在900 - 1100 mg/g 之间

ꔷ **木质活性炭**：

  **灰分**：一般在2% - 6% 之间。

  **硬度**：硬度相对较低，一般在60% - 75% 之间。

  **碘值**：一般在600 - 1000 mg/g 之间

ꔷ **煤质活性炭**：

  **灰分**：一般在5% - 15% 之间，具体取决于煤炭的种类和处理工艺。

  **硬度**：硬度较低，一般在50% - 75% 之间。

  **碘值**：一般在600 - 900 mg/g 之间

# 酸洗椰子壳：

**灰分：**可以到0.5%左右；

**硬度：**可能会略微降低，但基本上影响不大，因为椰壳活性炭本身具有较高的硬度。

## 单价：

不酸洗椰子壳1.2W/吨（即0.012元/克），含税

酸洗+0.25W/吨（即0.0145元/克）

# 活性炭用量计算：

**基本参数：**

*   水先通过“KDF55”再通过“椰子壳活性炭”
    
*   椰子壳活性炭：碘值为1200mg/g，灰分0.5%，比表面积1110m²/g，水分7.76%，表观密度0.49 g/ml
    
*   KDF55含量：204g
    
*   过滤水量：6768 GAL
    
*   预期使用寿命为：12个月
    

*   余氯浓度：2毫克每升（mg/L）
    

---

*   假设“KDF55”去除70%的余氯
    

计算结果：12.8g椰子壳活性炭，成本是0.1856元；
