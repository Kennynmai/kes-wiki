## 二、各平台详解

### 2.1 Reddit

| 项目 | 详情 |
|------|------|
| **注册方式** | 邮箱注册（推荐）或手机号注册 |
| **邮箱** | ⚠️ **技术上可选**——可以在注册时不绑定邮箱，7 天后再添加。但强烈建议用 @keshome.com 邮箱注册，以实现资产控制 |
| **手机号** | ⚠️ 可选。手机号注册可绕过部分风控检查，但非必需 |
| **真实姓名** | ❌ 不需要。用户名完全自由 |
| **头像** | ❌ 不需要。但建议设置（增加可信度） |
| **CAPTCHA** | 注册时可能触发 reCAPTCHA |
| **人类验证（2026 年 3 月新增）** | ⚠️ **仅针对被标记为可疑行为的账号**触发，非默认要求。触发条件：异常发帖速度、异常互动模式、自动化软件特征。验证方式：Passkey（Apple/Google/YubiKey）、生物识别（Face ID/Touch ID/World ID）、政府 ID（仅法律强制地区）。新号正常行为不触发 |
| **KES 准备清单** | `kenny@keshome.com` 邮箱（Google Workspace 已就绪） |

**参考来源：**
- Reddit 官方帮助中心（support.reddithelp.com）
- Reddit 2026 年 3 月人类验证公告（WinBuzzer, 2026-03-26）
- CSDN 注册避坑指南（2025-08）

### 2.2 Quora

| 项目 | 详情 |
|------|------|
| **注册方式** | 邮箱注册 / Google 账号 / Facebook 账号（三选一） |
| **邮箱** | ✅ 必填（如果用邮箱注册） |
| **手机号** | ❌ 不需要 |
| **真实姓名** | ✅ **建议遵守。** Quora 有「真名政策」（Real Name Policy），要求使用真实姓名。但无政府 ID 验证。格式建议：`First Name + 姓氏首字母`（如 `Kenny M.`）配合个人简介说明 KES 身份 |
| **头像** | ⚠️ 技术可选，但**强烈建议**上传真实头像（Quora 社区对无头像账号的信任度极低） |
| **逐答案凭据（Credentials）** | ⚠️ Quora 独有功能。每条回答可设置不同的「凭据」（如「KES 创始人 · 10 年卫浴五金」「产品工程师 · 阀芯测试」）。这是建立专业可信度的关键 |
| **KES 准备清单** | `kenny@keshome.com` 邮箱 + Kenny 真实头像 + 个人简介文案 |

**参考来源：**
- Quora 官方使用指南（wikiHow, 2025-07）
- Quora 中文完全指南（mingnify.com, 2025-08）

### 2.4 Slack

| 项目 | 详情 |
|------|------|
| **注册方式** | 邮箱注册（需工作区邀请） |
| **邮箱** | ✅ 必填。Slack 发送 6 位验证码到邮箱完成验证。同一邮箱可加入多个工作区 |
| **手机号** | ❌ 不需要 |
| **真实姓名** | ⚠️ 接受邀请时需要填写全名（Full Name），但无验证 |
| **加入方式** | **必须通过邀请。** 两种路径：① 直接输入邮箱→Slack 发验证码→选择工作区→加入；② 收到邮件邀请→点击「Join Now」→输入全名→加入 |
| **注意** | Slack 社区是邀请制的，必须先被社区管理员邀请才能加入。不能主动搜索和加入 |
| **KES 准备清单** | `kenny@keshome.com` 邮箱；需先确定目标 Slack 社区并获取邀请链接 |

**参考来源：**

### 2.5 Facebook（重点：Egypt 市场）

| 项目 | 详情 |
|------|------|
| **注册方式** | 邮箱注册 或 手机号注册（二选一） |
| **邮箱** | ⚠️ 与手机号二选一。如用邮箱注册，Facebook 向该邮箱发送验证码完成验证 |
| **手机号** | ⚠️ 与邮箱二选一 |
| **真实姓名** | ✅ **必填。** First Name + Last Name 均必填。可用英文名 |
| **生日** | ✅ **必填。** 月/日/年均需填写 |
| **性别** | ✅ **必填。** Female / Male / Custom（可选代词） |
| **头像** | ⚠️ 技术上可选，但**强烈建议上传**——无头像的新号加入群组时被拒概率高 |
| **身份验证（自拍）** | ⚠️ **新注册账号如果快速加入多个群组**，可能触发 Facebook 的自拍验证（需上传实时自拍匹配）。**建议：优先使用 Kenny 现有个人 Facebook 账号管理品牌页，而非新注册号** |
| **政府 ID** | ❌ 标准注册不需要。仅在账号被锁定或被黑客入侵时可能触发 |
| **品牌页（Facebook Page）** | 创建品牌页无额外资料要求，但需先有个人账号。品牌页的管理员权限可以转移，是公司可控的资产 |
| **KES 准备清单** | 方案 A（推荐）：用 Kenny 现有个人 FB → 创建 KES Egypt 品牌页；方案 B：新注册 `egypt@keshome.com` 个人号 → 30 天热身 → 创建品牌页 |

## 四、关键风险提示

### 4.1 避坑要点

| 风险 | 平台 | 预防措施 |
|------|------|----------|
| **新号快速加群触发自拍验证** | Facebook | 用 Kenny 现有个人号；新号必须先热身 30 天 |
| **同一 IP 多账号注册触发关联封禁** | Reddit | 不同账号用不同网络环境注册（不要在公司 WiFi 下同时注册多个号） |
| **新号 2 分钟内发帖被标记为 bot** | Reddit | 注册后至少等 30 分钟再发帖（CSDN 指南建议 7 天后再绑定邮箱） |
| **Quora 用假名被举报** | Quora | 用 First Name + 姓氏首字母 + 个人简介说明身份 |
| **Discord 用个人设备导致离职后数据残留** | Discord | 用公司设备或虚拟机登录 |
| **Slack 社区找不到入口** | Slack | 先确定 2-3 个目标社区，通过 KES 人脉或行业关系获取邀请 |

### 4.2 注册后不可逆的操作

| 操作 | 平台 | 后果 |
|------|------|------|
| 用个人邮箱注册 | 全部 | 离职后无法收回账号 |
| 用个人手机号绑定 2FA | 全部 | 离职后无法转移 |
| 用户名设置后不记录 | 全部 | 无法追踪 |

> **底线：一切注册必须通过 @keshome.com 邮箱 + Google Workspace TOTP。**
