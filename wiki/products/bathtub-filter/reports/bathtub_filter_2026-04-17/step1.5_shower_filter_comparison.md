## 3. 关键洞察

1. **Shower filter 是成熟红海**：top1 Aquabliss 的 click share 只有 27%（相比 bathtub 单词场景 60%+），说明流量分散，至少有 5-10 个可见玩家在竞争。
2. **Canopy 已在 shower filter 有存在（B0CJ3G16QZ）但很小**（只在 2 个词里出现）。它在 bathtub 维度的垄断可能是**在 shower 红海里找到的新细分切口**。
3. **品牌词搜索存在度**：Aquabliss、Jolie、Santevia、Aquahomegroup 都有专门的品牌搜索词 → shower filter 是**品牌化成熟品类**。Bathtub filter 只有 Canopy 和 Santevia 有品牌词 → **尚待品牌化**。
4. **细分场景高度重合**：shower filter 与 bathtub filter 核心诉求一致（除氯、硬水、皮肤/头发保护）。Shower filter 长尾词如 `shower filter for hair and skin`, `vitamin c shower filter`, `shower filters to remove toxic chemicals` 可直接反映 bathtub filter 未来会出现的诉求演化方向。

## 4. 战略意义

- **Bathtub filter 是 shower filter 的子集/衍生**：核心技术相同（KDF、活性炭、维生素 C、陶瓷球），只是安装位置不同（浴缸龙头 vs 花洒）。
- **Canopy 策略**：不和 Aquabliss / Jolie 在 shower 红海正面交锋，而是用**婴儿场景 + 3-in-1 产品设计**把浴缸这个细分做起来。
- **进入窗口**：Bathtub filter 还处于**早期**，但品类可能被 Canopy 的爆款教育带飞 → 如果 KES 要进，需要抢 2026 年内窗口期。
