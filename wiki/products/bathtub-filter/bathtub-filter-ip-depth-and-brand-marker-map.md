# Bathtub Filter 竞争 IP / 品牌标记深度地图

## 为什么有这页
这页不做法律意义上的 FTO，
而是回答更实用的问题：

> **哪些品牌看起来更像有真实技术 / 设计锚点，哪些更像主要靠 marketing posture（营销姿势）在卖？**

## 当前一眼判断
### 1. 这个类目的 IP 深度并不均匀
经 [2026-04-17 patent investigation workflow 验证过一轮](../../playbooks/patent-investigation-workflow.md)后，分四类：

#### A. historical media / chemistry lineage — 已全部过期
代表：**Farley family（David K. / Michelle / David Aaron Farley，Sprite CEO 个人及家族）**
- hot-water dechlorination
- calcium sulfite / copper-zinc（"Chlorgon"）
- shower-lineage chlorine media story

**修正：这条血统并不是"Sprite 这家公司"的 IP，而是 Farley 家族个人专利组合。** Sprite Industries 作为公司实体不持有这些专利，CHLORGON 的基础专利（`US5914043A` / `US6056875A`）2015 年底已过期，`US6056875A` 2001 年还曾抵押给 Chase Manhattan。Farley 家族后续又登记了几件现役专利（`US9504940B2` / `US10737203B2` / `US11458487B2`，2034–2040 到期），但都是以个人名义持有——死在这个人手上就结束。

对 KES 的含义：**技术血统**是真的，**公司护城河**是假的。

#### A'. 新晋 utility-patent-led 入场者 — 现役 IP 最猛
代表：**FilterBaby（FilterBaby LLC + Saxton Brothers LLC 共同持有）**
- 2026-01-27 授权 `US12534389B2`，2044 年到期
- claim 涵盖水平 faucet filter + 可溶解 **skin-treatment adapter** + 内置 **UV 杀菌** + 机械 filtered/unfiltered/eco 模式切换 + 智能监控
- 另持有设计专利 `USD1067376S1`
- 自称 "40+ global patents"、IAPMO NSF 177 认证
- 获 *TIME Best Inventions* 2024、2025 两次

这是目前唯一像 **utility-patent-led corporate moat** 的样本，且 claim 范围可能覆盖整个 DTC 护肤浴室过滤器类别。KES 若走 skincare 定位，必须先把完整 claim 读掉并做 design-around。

#### B. form-factor / configuration lineage 清楚，但 moat 不一定来自强功效证明
代表：**Crystal Quest bath-ball route**
- bath-ball 形态长期存在
- `D1034899` Bath Ball design patent 2024 授权——但**原申请人是中国跨境电商（泉州牛瑞力斯）**，2024-08-14 已转让给**余姚康普水处理设备厂**（中国 OEM），不是 Crystal Quest 本身

这类更像 **design / form continuity（设计与形态延续）**，且 design patent 在 Amazon 白牌卖家 / OEM 手里流动，并不天然形成西方 DTC 品牌的护城河。

#### C. brand-layer conversion 强，但 public patent anchoring 弱
代表：**Canopy / Santevia / Kinder / Tubo**
- route identity 很清晰
- 文案 / 设计 / segment 很强
- **经 2026-04-17 验证：这四家在本次扫描中都没有可见的 utility patent 挂在各自品牌名下。** Kinder Filter 网站宣称 "patented advanced filtration technology"，USPTO 记录里查无对应 Kinder 名下申请。
- 真正可见 IP 的新玩家里，**Homewerks Worldwide**（2022 年从 Waxman 买下 `US10159991B2`，2036 年到期）是唯一有公司级 utility patent 的下游 —— 但它是供应 Lowe's / Home Depot 的 supplier，不是 DTC 品牌。

这类仍是 **brand system moat**，不是 utility-patent moat。

## 2. 这个类目里要区分 3 种“护城河幻觉”
### Utility-patent-adjacent
通常对应：
- media chemistry
- hot-water chlorine reduction logic
- shower/tub cartridge architecture
- forced-flow / spout-filter configuration

这类更支持 **技术可信度**。

### Design-patent-adjacent
通常对应：
- bath-ball shell
- over-faucet shell form
- distinct exterior silhouette

这类更支持 **外观区分度**，
不自动支持 filtration claim credibility。

### Mostly marketing-language-led
典型表现：
- clinically tested
- patent pending
- 8-stage / 17-stage
- premium baby / eczema / safe-water theater

这类更支持 **转化氛围**，
不代表真正的技术壁垒。

## 3. 品牌逐个看
### Sprite（= Farley 家族 IP 的商业化载体）
当前最像：
- **individual-inventor-led**（修正：原 v1 写的 "technical-lineage-led" 方向正确，但主体错了——不是"Sprite 公司"有 IP，是 Farley 家族个人有 IP）

原因：
- 与 calcium sulfite / hot-water dechlorination patent lineage 邻接最强
- shower-lineage 与 narrower chlorine story 更一致
- 但 **`US5914043A` / `US6056875A` 已 2015 过期**，基础媒介专利已在公有领域
- 现役 Farley 专利（`US9504940B2` / `US10737203B2` / `US11458487B2`）都是 showerhead 架构，不是浴缸架构

结论：
- 仍是最值得 KES 参考的"技术收敛型样本"
- 但可参考的是 **story-telling 纪律**（窄技术血统 + 单一 media 命名），不是"去和 Sprite 打 IP 仗"
- 真做 calcium sulfite / copper-zinc 组合媒介，**法律上已经完全自由**

### FilterBaby（= 当前唯一像公司级 IP 护城河的样本）
当前最像：
- **utility-patent-led corporate moat（新类别，原 v1 漏掉）**

原因：
- `US12534389B2` 2026-01-27 刚授权，2044 到期
- claim 直接覆盖"skincare faucet filter"这个类别边界
- 有 TIME Best Invention 2024 + 2025 连续外部背书
- IAPMO NSF 177 认证

结论：
- KES 若做 skincare 定位浴室过滤，**这是必须绕开的一件专利**
- 具体要读：可溶解 skin-treatment adapter 的 claim language、UV 内置的 claim language、机械 mode lever 的 claim language
- 设计自由度仍在：**faucet 取向（垂直 vs 水平）、媒介配方、cartridge 可见性的表达方式**——这三处是 FilterBaby claim 的盲区
- 这是目前**比 Sprite 更值得当竞争对手**的对象

### Crystal Quest
当前最像：
- **form-factor + cartridge-configuration route**

原因：
- bath-ball route 长期可见
- 公开 surface 已出现 design-patent 线索
- 但并没有同步看到同等级别的窄功效护城河表达

结论：
- 比“只是营销”更强一点
- 但强项更像外形 / route continuity，不是最干净的 proof moat

### Canopy
当前最像：
- **design-first / segment-first moat**

原因：
- baby registry / wellness / premium object 逻辑非常清楚
- bath SKU 前台仍以设计、颜色、spout-cover、registry、gentle bath world 为主

结论：
- 它的 moat 更像品牌系统和 visual discipline
- 不是强 public patent signaling 样本

### Santevia
当前最像：
- **ritual / wellness object route**

原因：
- mineral / gentle nourishment / bath ritual story 明显
- hanging accessory 逻辑比 patent-anchor 更突出

结论：
- 更偏 wellness-led，不是 obvious patent-led

### Kinder
当前最像：
- **clinical-positioning DTC without strong visible IP anchor**

结论：
- 转化语言强
- 可见 IP 深度弱
- 更像用 positioning 去替代 moat impression

### Tubo
当前最像：
- **performance-claim-heavy DTC / marketplace hybrid**

结论：
- claim energy 很强
- 可见技术 / patent 锚点仍弱
- 更像 marketing-led growth posture

## 4. 当前更重要的不是"谁有专利"，而是"谁在借什么东西建立可信感"
目前看，可信感来自四种来源（原 v1 是三种，补一类）：
1. **历史技术 lineage（已过期）** — Sprite / Farley 血统
2. **现役 utility patent** — FilterBaby（`US12534389B2`），以及 Homewerks 在分销侧的 `US10159991B2`
3. **设计 / visual distinctiveness** — Canopy、部分 Crystal Quest（注意 bath-ball design patent 实际掌握在中国 OEM 手里）
4. **clinical / parenting / testimonial theater** — Kinder、Tubo 最明显；他们宣称的 "patented" 与可见 USPTO 记录不对应

这四种来源对 KES 的意义完全不同，也对应完全不同的应对策略。

## 5. 对 KES 的直接启发
### 不要误判
- 高转化 ≠ 高 IP 深度
- 有 design patent ≠ 有强性能护城河
- 没快速看到 patent ≠ 一定没有保护
- **"Sprite lineage" ≠ "Sprite 公司 IP"**（是个人 IP，已过期）
- 品牌自称 "patented X technology" **不能**当作 IP 存在的证据——需要回到 USPTO 验证具体 filing

### 更值得学的
- Farley 血统的**窄故事纪律**（单一 media 命名、narrower-chlorine-story）
- Crystal Quest 对 form factor continuity 的启发
- Canopy 的 design-system / segment packaging
- **FilterBaby 的入场姿势**：不靠老专利背书，直接建新 utility patent，再用 TIME Best Invention 做外部权威背书

### 更不该照抄的
- 用 clinically tested / baby-safe / eczema-forward 当作 moat 替身
- 用 patent-pending 氛围替代明确 route discipline
- **假借已过期的 Chlorgon 血统说自己"专利技术"**——这是误导，不是 moat

### KES 需要主动规避的专利（2026-04-17 现役名单）
必须绕开或 design-around：
- `US12534389B2` (FilterBaby，2044) — 若做 skincare 定位 **必读**
- `US10159991B2` (Homewerks ex-Waxman，2036) — split housing + reversible cartridge
- `US9504940B2` / `US10737203B2` / `US11458487B2` (Farley 家族) — 多为 showerhead 架构，对浴缸直接撞 claim 的概率较低

## 当前一句话结论
**旧的"Sprite 技术护城河"叙事在法律层已全部过期；真正的现役护城河集中在一件 2026 年刚授权的 FilterBaby 专利（`US12534389B2`，2044 到期）上。bathtub filter 赛道里其他品牌仍以 brand / segment moat 为主，而 KES 的主要 IP 对手从 "Sprite / Canopy" 应改读为 "FilterBaby"。**
