# 埃及翻译/管理人员候选人评估框架

## Trigger / scope

适用于 Kenny 在埃及招聘翻译、行政协调、基层管理岗位（未来可能晋升为团队负责人）的候选人评估。

目标不是找"中文最好的人"，而是找到：

**能长期跟着公司从0做到1的人。**

## 评估维度（权重排序）

### 1. 责任感（权重最高）
观察指标：
- 不明白时是否主动确认，而非装懂
- 是否能反复跑去问、查清楚再回来汇报
- 是否愿意承担"搞明白"的责任，而不只是"翻译"的责任

**判断标准**：确认型沟通 > 语言流利度

### 2. 诚实度（第二权重）
证据来源：
- 是否愿意接受背调，是否主动提供前领导联系方式
- 是否主动出示工资单/可验证的收入证据
- 面试中是否有明显夸大或前后矛盾

**重要区分**：老实 ≠ 可靠。老实可能是面试表现，可靠需要时间验证。

### 3. 自驱力与学习能力
观察指标：
- 中文/专业技能是否为自学（自学的含金量 > 大学专业）
- 是否表现出持续学习的意愿和行为
- 教过一次的事情，下次是否还犯

### 4. 稳定性意图
信号：
- 年龄阶段（埃及男性28-35岁进入成家求稳阶段）
- 是否明确表达"想要一份长期稳定的工作"
- 家庭情况（已婚/未婚，住所安排）
- 前工作更换频率

### 5. 管理潜力（加分项，非必须）
验证方法：
- 是否有真实的管理经验（非"班组长"级别）
- 用具体场景追问："60个人迟到怎么办？""两人打架怎么办？""工人离职怎么办？""中埃经理冲突怎么办？""如何安排加班？"
- 回答是否有逻辑和可操作细节

### 6. 中文能力（必要但不充分）
评估方式：
- **禁止仅靠面试判断**
- 必须带去医院、银行、Vodafone、政府部门等实战场景
- 核心问题不是"中文多好"，而是"能不能把老板的意思准确传达"

### 7. 行为细节（辅助判断）
值得注意的细节：
- 吃饭时的举止（放松状态下最真实）
- 对金钱的态度（是否贪小便宜、分寸感）
- 社交边界感

## 评估流程

### 第一阶段：面试（初筛）
- 背调意愿 → 工资单真实性 → 中文水平初判 → 管理经验追问
- 产出：初评分数 + 关键待验证项

### 第二阶段：实地试用（1-7天）
- 带到政府、银行、运营商等真实场景
- 观察沟通模式（确认型 vs 装懂型）
- 共进午餐，观察放松状态
- 产出：是否进入正式试用期

### 第三阶段：试用期（3个月）
正式观察五项：
1. **守时**：连续一个月，迟到次数和理由
2. **诚实**：不知道就说不知道，还是开始编答案
3. **学习能力**：教过的是否记住
4. **忠诚度**：公司困难时是否愿意一起扛
5. **管理潜力**：逐步给协调任务，观察表现

工资：20000埃镑/月，岗位：中文翻译

### 第四阶段：晋升评估（试用期后）
条件：
- 试用期五项观察全部通过
- 表现出管理意愿和能力
- 公司确实有管理需求

晋升路径：
```
翻译
↓
翻译 + 行政协调 + 工人管理
```

岗位津贴：5000~10000埃镑/月

## 红线和否决项

以下情况应直接否决：
- 拒绝背调或背调结果严重负面
- 面试中明显撒谎
- 工作经历严重造假
- 前雇主的负面评价涉及诚信问题

## 敏感问题的处理

### 身体情况（如狐臭）
- 不要在面试阶段评价
- 录用后私下单独沟通
- 沟通话术：强调公司形象要求，公司愿意协助
- 原则：不影响工作能力就不应成为录用障碍

### 金钱试探
- 不要因为第一次推辞就给高分
- 观察长期行为模式
- 区分"真的不好意思"和"社交客气"

## 候选人档案模板

每个重点候选人应建立独立档案：

```markdown
## 候选人 [编号/姓名]
- 面试日期：
- 初评分数：/100

### 基本信息
- 年龄、婚姻状况、住所安排
- 前雇主、前工资（有无证明）
- 中文水平及来源

### 加分项
- [ ] 接受背调
- [ ] 出示工资单
- [ ] 中文自学
- [ ] 管理经验
- [ ] 确认型沟通

### 待观察
- [ ] 中文实战水平
- [ ] 管理能力验证
- [ ] 长期稳定性
- [ ] 背调结果

### 跟踪记录
| 日期 | 观察内容 | 评分变化 |
|------|---------|---------|
| | | |
```

## Review notes

- 本框架基于候选人A的面试案例提炼，随着更多候选人面试应持续迭代
- "确认型沟通"作为核心指标是从本次案例中浮现的关键洞察
- 建议每季度回顾一次，根据实际用人结果校准各维度权重
