# Bathtub Filter — 特殊水源类型指南

## 为什么有这页
市政供水（游离氯 / 氯胺）和私人井水只覆盖了美国用水家庭的大多数，但还有若干重要的特殊水源类型，每种都有截然不同的水质特征，对产品的适用性和宣称边界有根本性影响。

本页回答：
> **各类特殊水源用户是否适合使用标准 bath filter？产品对他们有效、无效，还是有风险？**

---

## 总览矩阵

| 水源类型 | 美国家庭数量 | 主要洗浴水质问题 | 标准去氯 bath filter 是否有效 |
|------|------|------|------|
| **全屋软水器（市政供水）** | **1,400–2,200 万** | 氯/氯胺仍然 100% 保留，软水器不去氯 | ✅ **高度有效** — 最大未被服务细分 |
| 雨水收集系统 | 约 20–100 万 | 细菌、重金属、沉积物；无氯 | ❌ 无效——无氯可去除；真实风险未被解决 |
| 蓄水池 / 拉水（USVI、纳瓦霍等）| 约 15–50 万 | 大肠杆菌（80%）、耐格里虫（30%）；无氯 | ❌ 危险——无氯；产品无法应对实际风险 |
| 小型社区供水（有消毒）| 约 150–200 万户 | 游离氯（非氯胺）；偶发大肠杆菌 / 砷超标 | ✅ 游离氯去除有效；砷/细菌不覆盖 |
| 小型社区供水（无消毒）| 约 50–100 万户 | 细菌、砷、硝酸盐；无氯 | ❌ 无效——无氯；目标污染物不同 |
| 全屋反渗透（RO）| 约 20–50 万户 | 近零 TDS / 无氯；水质极纯；轻微酸性 | ❌ 无效——已无氯；产品无附加价值 |
| 波多黎各（PRASA 供水）| 约 100 万户 | 游离氯 + 高卤代烃（THMs）+ 铅 | ✅ 游离氯有效；THMs 部分有效；铅需专用滤材 |
| 夏威夷市政供水 | 约 40 万户 | 游离氯，火山岩软水 | ✅ 正常有效 |
| 夏威夷大岛雨水收集用户 | 约 6 万人 | 细菌、火山雾酸性水、重金属；无氯 | ❌ 无效 |
| 关岛（PFAS 污染区域）| 约 5 万户的子集 | 游离氯 + PFAS（军事基地污染）| ⚠️ 氯去除有效；PFAS 需 RO 等专项技术 |

## 1. 全屋软水器家庭（**最大未被服务细分**）

### 规模
- 约 **1,400–2,200 万**美国家庭安装了离子交换式全屋软水器
- 覆盖约 3,500–5,500 万人
- 集中在硬水地区：印第安纳、俄亥俄、密歇根、明尼苏达（这些州约 60% 的家庭安装了软水器）、德克萨斯、亚利桑那、内华达

### 软水器对水质做了什么、没做什么

| 参数 | 软水处理前 | 软水处理后 |
|------|------|------|
| 硬度（Ca²⁺/Mg²⁺）| 高（如 300 mg/L）| 近零（< 5 mg/L）|
| 游离氯/氯胺 | 完整保留 | **完整保留——软水器不去氯** |
| 钠（Na⁺）| 低 | 升高（与去除的硬度成比例）|
| 水垢 | 会形成 | 消除 |
| pH | 不变 | 基本不变 |
| DBPs（三卤甲烷）| 存在 | 完整保留 |

> **关键结论：软水器是单一目的设备——仅去除钙镁。氯 / 氯胺不受影响，100% 保留至浴缸中。**

### 对产品的核心含义

软水器家庭在洗浴时接触到的消毒剂浓度与未装软水器的家庭完全相同。**Bath filter 对这类家庭有明确、真实的价值，且与软水器功能完全互补，无冲突。**

**最有力的营销钩子：**
> "您的软水器去掉了水垢，但浴水里的氯仍然完整保留。"

已经花费 $500–$2,000+ 安装全屋软水器的家庭，对水质有高度敏感和投资意愿。他们很可能误以为软水器已经"解决了所有水问题"。

### 宣称指引
- ✅ "减少游离氯 / 氯胺"——完全有效
- ✅ "软水器不去除消毒剂，bath filter 补充这一功能"
- ❌ 不要说 bath filter "软化水"——对软水家庭毫无意义且不准确
- ❌ 不要说去除了软水器添加的钠——这需要 RO，不是滤材过滤

## 2. 雨水收集系统

### 规模与地理
- 大陆美国：估计约 20–100 万户（确切数字难以核实；常被引用的"20%"数字来源存疑，无法追溯到可信一手来源）
- 最集中的地区：**夏威夷大岛**（约 6 万人因火山岩地质无法打井，强度依赖雨水），部分德克萨斯、亚利桑那（法律积极鼓励），散布在山区 / 离网社区
- 德克萨斯、亚利桑那有税收优惠鼓励雨水收集；各州法律不一

### 雨水的水质特征
- **无硬度**：近零 Ca²⁺/Mg²⁺，天然软水
- **无氯**：未经市政处理，无任何消毒残余
- **pH 5.5–5.7**（微酸，溶解 CO₂ 形成碳酸）；工业区可低至 4.5（酸雨）
- **主要污染来源**：屋顶表面（沥青、涂料）、排水沟、储水罐
  - 大肠杆菌：同行评审研究约 18% 的屋顶收集水样检出
  - 全大肠菌群：更高比例检出
  - 铅（旧屋顶金属材料、含铅涂料）
  - PAHs（沥青瓦屋面）
  - 昆虫 / 鸟类 / 鼠类粪便污染
- 夏威夷大岛特殊情况："火山雾"（vog）含二氧化硫，增加酸性，可从屋顶金属材料浸出铅、铜、锌

### 对产品的含义
- **无氯可去除**：标准去氯媒体（KDF-55、亚硫酸钙、维生素 C）完全没有施展空间
- 实际风险（细菌、铅、PAHs）需要完全不同的技术：陶瓷/超滤膜、UV 消毒、活性炭（去 PAHs/VOCs）
- **严重宣称风险**：若将标准 bath filter 定位为雨水家庭的"水质改善"产品，可能造成错误安全感

> **结论：不要将标准 bath filter 营销给雨水收集用户。产品无法解决他们的真实问题，且有误导风险。**

## 3. 蓄水池 / 拉水家庭

### 规模与分布
| 群体 | 家庭数量 |
|------|------|
| 纳瓦霍族（亚利桑那 / 新墨西哥 / 犹他）| 约 1.4–1.5 万户拉水家庭（纳瓦霍族 ~30% 无管道供水）|
| 阿拉斯加偏远社区 | 约 3,300+ 户无室内管道供水 |
| 美属维尔京群岛 (USVI) | 约 3.86 万户（约 95% 依赖屋顶蓄水池）|
| 其他（阿巴拉契亚农村、美墨边境 colonias）| 数万户 |

### 水质状况（以 USVI 为例，最完整数据集）
2022 年同行评审研究（ACS ES&T Water，95 个 USVI 蓄水池样本）：
- **全大肠菌群**：95% 的水池检出
- **大肠杆菌**：80% 检出
- **隐孢子虫**：36% 检出
- **沙门氏菌**：32% 检出
- **耐格里虫（脑吃阿米巴）**：30% 检出
- **游离氯**：仅 48% 检出（中位值 0.03 mg/L，远低于 WHO 安全最低值 0.2 mg/L）

94% 的 USVI 家庭用蓄水池水洗浴——洗浴是最主要的暴露途径。

### 对产品的含义
- **最严重的宣称风险类型**：耐格里虫可通过洗浴时的鼻腔接触传播，致死率近 97%（原发性阿米巴脑膜炎）
- 标准 bath filter 完全无法处理细菌、原虫或阿米巴
- 若将产品定位为这类家庭的"水质改善"方案，属于严重误导，存在法律和道德风险

> **结论：蓄水池 / 拉水家庭是禁止营销区域（排除在目标用户画像之外）。**

## 4. 小型社区供水系统（有消毒）

### 规模
- 美国约有 14.8 万个公共供水系统，其中 97% 是"小型系统"（服务 ≤1 万人）
- 服务 25–3,300 人的系统约 4.3 万个，覆盖约 **150–200 万户家庭**
- 集中在：中西部和大平原农业社区、阿巴拉契亚农村、美国东南部农村、部落土地、山区小镇

### 与大型市政系统的区别
- 同等 EPA SDWA MCL 标准适用，但执行能力差异巨大
- 有消毒的小型系统几乎全部使用**游离氯**（不是氯胺）——因为氯胺需要精密运营能力
- 合规记录更差：EPA 执法优先名单中 **93% 是小型系统**；PNAS 研究显示每年 7–8% 的小型系统有健康相关违规
- 许多有违规的系统三年后问题依然存在（慢性而非偶发）

### 对产品的含义
- **游离氯去除**：完全有效；小型有消毒系统与标准市政供水无本质区别
- **砷**：局部高风险区域（西部、新英格兰）；标准 bath filter 不去除砷
- **细菌穿透事件**：偶发；产品不处理微生物
- **底线**：约 150–200 万户可用标准产品宣称；但应避免对这一细分声称"改善整体水质"

## 5. 全屋反渗透（RO）家庭

### 规模
- 全屋 RO（入口处理）：约 **20–50 万户**（主要在西南部和山区西部，高 TDS / 高砷井水地区）
- 水槽下 POU-RO（仅饮用水）：约 1,500–2,000 万户（但这不改变浴缸用水）

### 全屋 RO 出水化学特征
- **TDS**：近零（进水 2,200–8,485 ppm → 出水 52–188 ppm）
- **硬度**：近零（所有钙镁去除）
- **游离氯 / 氯胺**：99%+ 去除——**浴水中已无消毒剂**
- **pH**：微酸（5.5–6.5），无矿物质缓冲
- **矿物质**：全部去除
- **铅、砷、PFAS、DBPs**：高效去除

### 对产品的含义
- 标准去氯 bath filter **无价值**：已无氯可去除
- 可能的小众定位：
  - UV 微生物保护（防止管道生物膜蔓延至出水口）
  - 矿化补充（极纯水的"空洞感"——重新添加矿物质）
- 这是一个技术复杂、规模小的细分；不是标准产品的目标用户

## 6. 波多黎各

### 规模与背景
- 约 **110 万户**家庭（人口约 320 万）
- 97% 由 PRASA（波多黎各供水管理局）服务
- 消毒剂：**游离氯**（非氯胺）
- **99.5% 的波多黎各人口由有违规记录的供水系统服务**（NRDC 数据）
- 最常见违规：铅铜规则（97.2% 人口覆盖系统有违规）、三卤甲烷 / 卤乙酸（热带气候高有机物负荷 + 氯化 = 高 DBP）

### 飓风对水质的影响
2017 年玛丽亚飓风后文献（至 2024 年）：
- 230+ 万居民受大肠菌群检出供水系统影响
- 砷、苯并三唑、PFOA、铅浓度飓风后上升
- 基础设施受损导致水质在"正常市政供水"和"事件后污染"之间波动

### 对产品的含义
- **游离氯去除**：完全有效（与大陆游离氯城市相同）
- **DBPs（三卤甲烷）**：热浴水中会挥发，部分可被活性炭滤材处理——是比大陆城市更强的购买动机
- **铅**：需要专用铅去除媒体；标准 KDF-55 有有限铅去除能力，但非其主要功能
- **市场评估**：约 110 万户家庭，直接可通过美国电商（Amazon.com）触达，双语（英/西班牙语）包装。购买驱动力强于大多数大陆城市。**值得作为二级市场认真对待。**

## 7. 对 KES 产品策略的整体含义

### 可用标准产品服务的水源
（标准去氯配方有效，宣称可站得住）

1. **全屋软水器家庭**（1,400–2,200 万户）：最大被忽视的细分，氯仍完整保留
2. **标准市政游离氯城市**（主要分析在 water-jurisdiction 页面）
3. **标准市政氯胺城市**（需维生素 C / 催化活性炭配方）
4. **小型有消毒社区供水**（约 150–200 万户游离氯）
5. **夏威夷市政供水**（游离氯，正常有效）
6. **波多黎各**（游离氯；DBP 故事比大陆更强）

### 必须从产品营销中明确排除的水源
（产品无法解决真实问题，或有误导风险）

- 雨水收集系统（无氯；细菌/重金属是主要问题）
- 蓄水池 / 拉水家庭（无氯；致命微生物风险无法通过 bath filter 解决）
- 无消毒小型供水系统（无氯；错误工具）
- 全屋 RO 家庭（已无氯；产品无附加价值）

### 软水器家庭：最值得开发的新定位角度

| 当前大多数 bath filter 的定位 | 软水器家庭专属定位 |
|------|------|
| "去除浴水中的氯" | "您的软水器去了水垢，但氯仍然在浴水里——bath filter 补全这一环" |
| 硬水受害者 | 已经投资水质、最有行动意愿的用户 |
| 问题未被注意 | 水质敏感度已被软水器激活 |
