# KES Bathtub Filter · 水质自测套件 + 模块化获客引擎（Q3，优化版）

> 把「滤材组合 + cross-sell + 推荐朋友」做成 KES 的**主获客方式**。为「防止买错/组错」设计的**前端水质判断**，正好是这台引擎的入口——**同一个东西，一面防呆，一面获客**。
> 而「自由组合 + 透明 + 自己测」三个产品特征本为**诚实**而设，却恰好是最好的**参与、复购、转介绍**机制——**诚实和增长在此是同一件事。**

---

## 一、两条铁律（贯穿全篇）

1. **工具配参数 + 标注置信**：不同水质参数，最准的工具不同；每个判断都明说「有多确定」，绝不让粗筛冒充实验室。
2. **诊断 = 处方，但处方各自有界**：诊断结果直接映射到该买的模块；但每个模块的声称**各自如实**（去氯 ✅ / 软化 ❌ / 阻垢只保设备 / 重金属只是料级），否则就成了我们批 Tubo 的 claim 堆叠。

---

## 二、工具 × 参数 × 置信（优化后的判断矩阵）

| 要判断的 | 最该用的工具 | 置信 | 诚实边界 |
|---|---|---|---|
| **消毒剂类型（游离氯 / 氯胺）** | **ZIP → 水务 CCR 查询** | 🟢 高（权威） | KES 已有 [按 metro 水务图谱](./bathtub-filter-utility-service-map-by-metro.md)。这是类型的**权威判定** |
| 同上的**家庭自测交叉验证** | 试纸**游离氯 + 总氯**双垫 | 🟡 中 | **总氯 − 游离氯 = 结合氯（氯胺）**。若总氯≫游离氯→很可能氯胺。**减法放大误差，只作 flag，类型仍以 ZIP 为准** |
| 游离氯**浓度** | 游离氯试纸垫 | 🟡 中 | 粗筛档位足够（决定去氯负荷/寿命） |
| **硬度** | 试纸色块分档；要数字→滴定小瓶 | 🟡 中 | 给软/中/硬/极硬档；CCR 也有基线硬度可交叉 |
| **铁** | 试纸（粗）+ 锈染观察 | 🟠 中低 | 低浓度试纸不稳，锈黄染色症状更可靠 |
| **硫化氢/硫味** | 气味指引（臭鸡蛋味） | 🟠 症状 | 嗅觉信号，无试纸 |
| **沉积物/浊度** | 肉眼清澈度 | 🟠 症状 | 看得见的颗粒不需仪器 |

> **新增要点（来自氯胺研究）**：总氯试纸（Industrial Test Systems 等做 0.02–750 mg/L 游离+总氯）让用户**在家就能 flag 氯胺**，作为 ZIP 查询的交叉验证；它也正是氯胺版「买后验证」需要的那种试纸（见 §五）。

---

## 三、两条诊断路径（按认知负担分层，解决「品类认知近零」）

品类认知近零（Reddit 信号）→ 入口必须**极低摩擦**。给两条路，用户自选深度：

### 路径 A — 快速（30 秒，多数市政用户够用）
**只输 ZIP** → 立即得：消毒剂类型（权威）+ 基线硬度 + 推荐主配方。
> 对游离氯城市的大多数人，这一步就能给出「游离氯版 + 是否加阻垢」的可靠建议，**不需要任何试纸**。

### 路径 B — 深度（想确认/复杂水/井水）
**ZIP + 多垫试纸 + 观察指引** → 确认类型、量化浓度档、抓硬度/铁/沉积/氯胺 flag → 完整组合处方。
> 井水、老房子、敏感肌想精挑、或 ZIP 数据异常时走这条。

**默认推路径 A**（降摩擦），把路径 B 作为「想更准」的升级——而不是默认让所有人 DIY 拼滤芯。

---

## 四、读卡 = 模块处方（决策逻辑 + 有界声称 + 诚实叠加）

```
检出条件                →  推荐模块                →  声称边界（写死在读卡上）
──────────────────────────────────────────────────────────────────────
ZIP=游离氯 / 游离氯高     →  去游离氯主芯（CaSO₃+KDF55）   →  指定流量下去游离氯
ZIP=氯胺（总氯≫游离氯）   →  催化炭芯 + 抗坏血酸钠浸泡件 →  注水+浸泡阶段中和氯胺（按 4–8 分钟规划）★
硬度 极硬               →  阻垢模块                 →  「保护龙头/缸不结垢——我们不软化你的水」
铁 / 锈染               →  井水 KDF85 模块          →  低中度铁/硫味缓解（高铁>2–3ppm 劝退）
看得见颗粒              →  强化物理层（PP）          →  颗粒拦截、稳定水路
```
★ 氯胺接触时间见 [氯胺证据页 §9](./bathtub-filter-chloramine-media-research.md)；V1.5 上线前不主动推。

**诚实叠加规则（防 claim-stacking）：**
- 检出多条件 → 可推多模块，但**逐条说边界**，不打包成一句大词。
- **置信低的条件用「建议确认」措辞**，不用「你的水有 X」的断定（呼应 §二置信分层）。
- **能力外的诚实劝退**：高铁、杀菌、除砷/硝酸盐、软化 → 明说「compact 浴缸滤芯做不到，这是会有用的方向」，不卖错。

## 五、全生命周期：一套动作贯穿获客→信任→复购→增长

| 阶段 | 动作 | 工具 | 作用 |
|---|---|---|---|
| **获客** | 「先测你家的水」 | ZIP（路径 A）/ 试纸（路径 B） | 低摩擦钩子，踩中「在家自测」+「自己看见」 |
| **交叉销售** | 诊断→多模块处方 | 读卡（§四） | 诚实诊断让 cross-sell 不像 upsell |
| **买后验证** | 装芯前后各测一次 | **匹配配方的验证试纸** | 「测得到/自己测」的证明 |
| **复购触发** | 试纸告诉你该换了 | 验证试纸 | 不靠黑箱倒计时 lock-in，靠「你自己看到该换」 |
| **转介绍** | 和朋友一起测 | 自测套件 | 结果可分享，奖励挂在下一支滤材上 |

**验证试纸必须匹配配方（优化要点）：**
- **游离氯版** → 游离氯试纸验证（装芯前后游离氯下降）。
- **氯胺版** → **总氯试纸**验证（游离氯试纸测不出氯胺）。
- ⚠️ **抗坏血酸干扰**：刚加完维C/抗坏血酸钠立即测，残余还原剂会让读数偏低、高估效果——说明书须提示「按指引时机测」，否则用户自测会误判（同 §9 比色干扰）。

## 六、模块目录（被交叉销售的对象，各自边界如实）

| 模块 | 解决 | ✅ 可说 | ❌ 不可说 |
|---|---|---|---|
| **去游离氯主芯**（CaSO₃+KDF55） | 游离氯 | 指定流量下游离氯去除 | 氯胺 / 软化 / 重金属健康 |
| **催化炭芯** | 氯胺(inline 段) | 氯胺部分去除（需配浸泡件） | 单芯全除氯胺 |
| **抗坏血酸钠浸泡件** | 氯胺(缸内停留段) | 注水+浸泡中和结合氯（4–8 分钟） | 「快速/秒解」/ 护肤疗效 |
| **阻垢 adjunct** | 设备水垢 | 保护龙头/缸不结垢 | **软化水 / 护肤** |
| **井水 KDF85 模块** | 低中度铁/H₂S/味 | 铁锈/硫味 nuisance 缓解 | 杀菌/除砷硝酸盐/高铁全屋替代 |
| **强化物理层（PP）** | 沉积/颗粒 | 颗粒拦截、稳定水路 | 主过滤 KPI |

## 七、转介绍机制（机制 + 实际落地）

### 机制概览
- **可分享的诊断结果**：诊断生成一张「你家的水」结果卡（类型/硬度/建议组合），本身是可发社媒的内容资产（也喂 SEO/GEO 的「what's in your water」长尾）。
- **一起测**：送朋友一份自测套件 = 低成本破冰；teardown 钩子（「你说得出你家滤芯里有什么吗？」）天然可分享。
- **奖励挂在复购上**：referral 奖励兑换在**下一支滤材/模块**，把转介绍直接接进复购飞轮（而非一次性折扣）。

### 7.1 核心病毒载体：个性化「你家的水」报告卡
诊断完（ZIP 或试纸）自动生成可视卡：城市/ZIP、消毒剂类型、硬度档、推荐组合。
- **每张卡 = 一个带推荐人码的唯一链接**（`yourwater.keshome.com/r/xxxx`）+ 可下载图（IG/短信）。
- 朋友点进 → 落到「也测你家的水」流程 → 下单则推荐人得奖励。
- **关键**：转介绍链接**就是**一份有用的诊断结果，不是裸的「推荐有礼」——这是它能转起来的原因。

### 7.2 触发时机（验证峰值是最强点，别错过）
| 触发点 | 文案方向 | 强度 |
|---|---|---|
| 诊断后 | 「分享你的水报告」 | 中 |
| 拆箱卡 / 订单确认邮件 | 「测测朋友家的水」（QR） | 中 |
| **★ 自测验证后**（装芯前后看到氯降下来） | 「你刚亲眼看到它有用——分享给也用同样水的人」 | **最高（信任峰值）** |

→ **把 referral CTA 埋在「验证成功」那一屏/那封邮件**——用户刚自证产品有效，是最愿意分享的一刻。

### 7.3 激励结构（两端 + 挂复购）
- **推荐人**：奖励兑换在**下一支滤材/模块**（下次换芯 $X off，或满 N 人送一支）——挂复购不挂现金，同时拉 LTV/留存。
- **朋友**：首单 starter 折扣 或 免费诊断套件。
- 物理「一起测」：买家货里多附 1–2 张试纸卡（COGS 几毛）递给朋友；或数字端「送朋友一次免费水测」→ KES 寄免费试纸卡（带码）。免费试纸 ≈ $0.3 + 邮费，**远低于付费 CAC**。

### 7.4 地理协同（接首发 P1）
转介绍在**同水质聚集区**最强：邻居同一水务 → 朋友水报告大概率相似 → **「你邻居家也是同样的硬+氯水」**是极强社会证明。首发圈定拉斯维加斯/凤凰城/圣安东尼奥，**让转介绍在同 metro 内发生**，相关性与转化最高（接 [V1 地理](./bathtub-filter-kes-v1-pricing-channel-launch-geo-subscription.md)）。

### 7.5 技术栈（现成件，不自研）
- 转介绍：Shopify/Woo referral app（Friendbuy / Refersion / Yotpo Referrals）管发码、追踪、奖励兑付。
- 诊断 quiz：quiz app 或自建 calculator，ZIP→水质数据接 [水务图谱](./bathtub-filter-utility-service-map-by-metro.md)，结果页直接挂推荐商品（cross-sell）。
- 报告卡：动态 OG 图生成（serverless 每份渲染）。
- 追踪：GA4（已在栈）+ referral app + UTM。

### 7.6 分期落地
| 期 | 范围 | 工期 |
|---|---|---|
| **MVP** | referral app + 静态「分享你的水类型」链接 + 下次换芯抵扣 | ~2–4 周 |
| **完整版** | 动态个性化报告卡 + 寄免费试纸 + 验证峰值多触点触发 | MVP 验证后 |

### 7.7 护栏 + 指标
- **护栏**（继承全篇红线）：报告卡/分享文案不 toxin-panic、不健康声称、不软化承诺；framing = 「帮朋友搞清自己家水里有什么」，不是「赚钱刷朋友」；不绑定夸大宣称、不制造刷屏压力。
- **指标**：referral 分享率、k-factor（邀请×转化）、referral CAC vs 付费 CAC、**奖励→复购兑现率**（验证转介绍是否真拉复购）。

## 八、红线（否则变成我们鄙视的对手）

- ❌ **不夸大诊断精度**：试纸是粗筛指路，说「帮你找到对的组合」，不是「检测你水里的危害」；置信低的明说「建议确认」。
- ❌ **诊断不导向 toxin-panic**：读数说「水的类型/产品适配」，不是「危害你」。
- ❌ **cross-sell 不变 claim-stacking**：每模块边界如实，多条件逐条说。
- ❌ **诚实劝退而非卖错**：能力外（高铁/杀菌/软化/氯胺未上线）明说不适合 + 留邮箱/指方向。
- ❌ **不用 TDS 笔**（测总溶解固体，非氯非硬度，常被用来伪精确恐吓）。
- ❌ **不做铅/重金属试纸**（低浓度不可靠 + 非我们声称）。

---

## 九、供应链路径

定制多垫试纸是成熟私牌生意——**Industrial Test Systems (Rock Hill, SC) 正是做游离+总氯试纸（0.02–750 mg/L）的 OEM**（AWWARF 脱氯指南点名），印证路径可行。定制点 = **选反应垫（游离氯/总氯/硬度/铁/pH）+ KES 品牌色卡 + 「读数→模块」决策图例**，化学不自研，MOQ 不高。
> **真正的 IP/壁垒在「读卡映射设计 + ZIP 数据接入」,不在试纸本身。** 抄试纸容易,抄「把诊断做成诚实处方 + 低摩擦获客 + 复购飞轮」的体系难。
