# KES Bathtub Filter 媒体 / EE-A-T 策略 + Clean-Formula 感性叙事

## 为什么有这页

首要价值已定为「**看得见·测得到的诚实过滤**」（见 [定位决策页](./bathtub-filter-kes-positioning-and-problem-layer-decision-2026-06-15.md)）。这页讲**怎么把它讲出去**，分两版：

- **理性版（EE-A-T / 媒体纪律）**：怎么用可核查的证据赢得信任、打对手的「EEAT 剧场」
- **感性版（Clean-Formula 叙事）**：怎么用「干净配料 / 干净配方」的消费者直白需求，让普通人**在乎**

> **关键认知：两版不是两个定位，是同一根脊柱的两种语言。** 理性版对工程师 / 查证者说「测得到」；感性版对普通消费者说「看得懂、没隐藏」。**透明滤盒（产品定义 1.6）同时服务两者**——它既是第三方可核查的物理证据，也是 clean-label 情感的化身。

---

# 第一版 · 理性：EE-A-T / 媒体纪律

## 核心判断：不比「EEAT 剧场」，用「可核查」打「EEAT 剧场」

对手赢在剧场——FilterBaby 的 4 人皮肤科顾问团、N=35 赞助临床、AHLA「approved」自报印章、in-progress 认证说成 Certified；Canopy 的美学 + 达人种草。但底子薄、不可核查（见 [FilterBaby 案例](../../brand-studies/filterbaby-dtc-case-study.md)）。

KES 的结构性优势相反：**自有实验室 + 真认证 + 透明滤盒 + 第三方 DPD**（brand EEAT roadmap 审计：自有 lab 是 ⭐⭐⭐⭐⭐「绝大多数 DTC 对手没有」）。所以**用 substance 打 theater**，把首要价值延伸成四维「字面可查」：

| 维度 | bathtub filter 专属打法 | 打对手哪个弱点 |
|---|---|---|
| **Experience 经验** | 把 25L/min 去氯台架测试（[spec](./bathtub-filter-25lpm-dechlorination-bench-test-spec.md)）做成「我们怎么测」过程内容；逐级进出口测试视频；透明盒「看得见」= experience 视觉化 | FilterBaby N=35 是**买来的临床**；KES 是**自己测的一手经验** |
| **Expertise 专业** | 把「为什么按水源分配方」做成教育：游离氯 vs 氯胺化学差异、为什么 KDF 不是去氯主力（EBCT 太短）、为什么阻垢≠软化 | 对手靠**模糊化扩大宣称**；KES 用「诚实讲清边界」反衬专业 |
| **Authoritativeness 权威** | 只选**真第三方**：NSF/ANSI 42 材料端（KDF55 真实继承）、若做成品则 WQA Gold Seal；口径如实本身=权威差异 | FilterBaby 把 in-progress 说成 certified → **KES「我们没有 derm board，我们有第三方实测报告」** |
| **Trustworthiness 可信** | 透明盒=可信物理化身；赠氯试纸=验证权交给用户；逐级报告公开；**主动声明「不去重金属、不软化、不治湿疹」** | Canopy 站内 4.6 vs 站外 BBB F + 霉变落差 → **KES 用「主动说不能做什么」做最高级 trust** |

## YMYL 张力：bathtub filter 的 EEAT = 「做满 + 划界」

brand-level roadmap 可放心讲「安装/漏水/寿命」。但 bathtub filter 是 **YMYL-adjacent（皮肤/婴儿）**，碰皮肤健康即进医疗红线（A-10）。纪律：

> **在「水质/工程」维度做满（去氯率、料级、兼容、寿命，全部可核查），在「皮肤健康」维度克制（只做教育性因果链，绝不做疗效）。划界本身是信任资产。**

## 媒体打法（品类认知近零的特殊约束）

Reddit 信号：用户知道「洗澡后皮肤痒」，不知道「过滤器」是解。所以：

1. **问题先行，不等搜索**：从「为什么洗澡后皮肤干/氯味」教育因果链（氯/硬水 → 不适 → 过滤），严守不越界湿疹治疗。
2. **helpful-first 社区**（已有 RedLeads/Reddit 资产）：r/eczema、r/Parenting 真实回答，纪律=教育水质、不推销疗效。
3. **设定裁判标准的 teardown 内容（最锋利一招）**：做「**如何辨别一个 bathtub filter 的宣称经不经得起查**」——教用户看透明料、查第三方 DPD、核认证口径。**把整个品类的评判标准往 KES 强、对手弱的方向设定**。

# 第二版 · 感性：Clean-Formula 叙事

## 消费者洞察（张力）

现代消费者已被训练成「**读配料表的人**」：
- **食品 → clean label**：看配料表、拒绝看不懂的一长串添加剂、"short ingredient list"、"nothing hidden"
- **护肤 → clean beauty / skinification**：看成分表、"full ingredient transparency"、"你抹在脸上的东西很重要"

但 **洗澡水是最大的盲区**：你每天让全身（和宝宝）泡在里面 20–40 分钟，却从不知道里面有什么、用什么滤的。**滤芯是个黑箱，一锅看不见的杂豆。**

> **核心张力：你对吃进嘴、抹上脸的东西如此讲究——为什么对包裹全身皮肤的水，毫不设防？**

## 情感领地

**干净 / 透明 / 没有隐藏 / 看得懂 / 诚实**——**不是**「健康疗效」，是「干净诚实」。这与 clean-label 运动的情感内核完全一致：clean label 卖的也不是「治病」，是「**安心 / 不被骗 / 掌控感 / 被当成聪明人尊重**」。

## 为什么这个叙事对「这个产品」成立（不是套话）

| clean-label / clean-beauty 概念 | KES bathtub filter 的物理对应 |
|---|---|
| "读得懂的配料表" | **透明滤盒看得见真料真量**——从「读标签」升级到「**亲眼看见**」，比 clean beauty 还硬 |
| "no cheap fillers / no hidden nasties" | **metal-first 高级铜锌合金**，看得见的好料，不用藏起来的便宜杂豆 |
| "personalized / 为你定制的干净配方" | **按你家水源配方**（游离氯/氯胺/井水） |
| "skinification 成分护理" | **洗澡水的成分护理**——把护肤的「成分意识」迁移到包裹全身的水 |

> clean beauty 只能在标签上**印**成分；KES 能让你**真的看见**真料真量。这是「clean formula」从「读」到「看」的升级——**这是这个叙事真正的护城河，不是借来的概念，是产品本来就有的事实。**

## Hero line 候选（待测试）

- 「你读食品的配料表，你读护肤的成分表。你的洗澡水呢？」
- 「看得见的滤料，读得懂的配方。」
- 「别人的滤芯是一锅看不见的杂豆汤；KES 是一份你看得懂的配料表。」
- "Clean water care. Nothing hidden."
- "Skincare for your bathwater."（把 skinification 迁移到洗澡水）

## 语气（Tone）

- **直白、温和的觉醒感、掌控感**，尊重消费者智商（"你已经在读配料表了，我们只是让洗澡水也变得可读"）
- **不吓唬**——绝不走 toxin-panic

## 感性版红线（情感 + 皮肤 + clean = 最容易滑坡，必须守）

- ❌ **不许 toxin-panic / 恐吓**（"毒水伤害你的宝宝"）——反向越界，且违背温和觉醒 tone
- ❌ **"干净/clean" 不许暗示「去除有害物 = 健康/治皮肤」**——clean 讲的是**透明与料级诚实**，不是 efficacy；「干净的配方」≠「治好你的皮肤」
- ❌ **不许用 "chemical-free" / "toxin-free"** 这类伪科学绝对化（自来水含氯是合法消毒，不能 demonize 氯本身）
- ✅ 安全措辞：「看得见的真料」「读得懂的配方」「没有隐藏的杂料」「你已经在读配料表了，我们让洗澡水也可读」

# 两版如何协同

| | 理性版 | 感性版 |
|---|---|---|
| 对谁说 | 工程师 / 查证者 / 高决策成本买家 | 普通消费者 / 已有 clean-label 习惯的人 |
| 一句话 | 「测得到」 | 「看得懂、没隐藏」 |
| 共用物理证据 | **透明滤盒**（既是第三方可核查证据，也是 clean-label 情感化身） | |
| 共同红线 | 不碰皮肤健康疗效（A-10）；认证口径如实；可见≠有效要分开讲 | |

> **落地建议**：感性版做**首屏情感钩子 + 社媒/视觉**（解决「在乎」与认知唤醒）；理性版做**产品页证据模块 + EEAT/GEO 资产 + teardown 内容**（解决「相信」与查证）。先用 clean-formula 让人停下，再用可核查让人下单和复购。
