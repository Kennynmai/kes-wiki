## 术语约定
### Route type（路线类型）
- **Soft-hanging（软悬挂）**：悬挂在水流下方的柔性 / 织物袋形态
- **Bath-ball（浴球）**：卡接到浴缸龙头上的硬壳球形 / 球灯形态
- **Premium-spout（高端龙头）**：以设计主导的整体式浴缸龙头装置
- **Legacy-media（传统滤材）**：继承淋浴过滤器的滤材叙事与品牌逻辑
- **DTC-sensitive-skin（DTC 敏感肌）**：直接面向家庭 / 湿疹 / 婴儿语境的 DTC 品牌

### Posture（宣称姿态）
- **Narrow（克制）**：以去氯 / 气味 / 水感为主，不越边界
- **Mixed（中性）**：主打去氯 + 部分更广 claim，但不触及皮肤结果
- **Aggressive（激进）**：进入皮肤结果 / 婴儿安全 / 湿疹这类更广的宣称区间

## 品牌点评

### Santevia
本 cluster 最反常的案例：**宣传克制、性能反而强**——而不是"承诺最大的品牌"。这让它成为"低 claim 风险 × 高可信度"象限的参照。

### Sprite
最耐读之处：它的浴缸路线继承了**比许多更年轻的 bathtub-filter-first 品牌更清晰的滤材 / 技术血统**。典型的 legacy 品牌优势。

### Crystal Quest
仍然重要是因为 **bath-ball 作为消费者认知形态** 的持久性，而不是因为产品本身。但营销广度与独立评测之间的落差已经大到足以形成品类信任税的一部分。

### Canopy
**premium design + newborn channel** 的最清晰参照；也是"**美学定位跑在性能证据前面**"的案例。

### Kinder / Tubo
不是因为技术故事最清楚而重要，而是因为它们展示了这个品类可以多激进地向 **敏感肌 / 婴儿 / 湿疹情感区** 销售——成为 KES claim-boundary 设计时要对标的**反面参考**。
