# 混合动力小型软水器 - 技术设计方案

**创建日期**: 2026-06-14  
**设计状态**: 概念设计  
**目标产品**: 台面/壁挂式小型软水器（浴缸场景优化）  
**设计理念**: 可靠性优先，智能化可选  

---

## 1. 设计理念与核心价值

### 1.1 核心设计哲学
**"永远有软水用"** - 基本功能不依赖电力，智能功能锦上添花

### 1.2 用户价值主张
1. **可靠性第一**：停电、断电不影响基本软化功能
2. **安装友好**：无需预留电源插座，适合改造市场
3. **选择自由**：用户可以选择纯机械模式或智能模式
4. **维护灵活**：手动再生或电动辅助再生，用户自主选择

### 1.3 目标用户画像
- **主要用户**：城市公寓住户、浴室无电源插座、停电频繁地区
- **次要用户**：重视可靠性的用户、不喜欢复杂电子设备的用户
- **价格敏感用户**：希望以较低成本获得基本软化功能

### 2.2 运行模式

#### 模式A：纯机械模式（基础版）
- **电源需求**：无
- **功能**：
  - ✅ 正常供水软化（机械阀控制）
  - ✅ 手动再生（手动手柄操作）
  - ❌ 无智能显示
  - ❌ 无自动再生
- **适用场景**：无电源、预算有限、简单需求

#### 模式B：智能辅助模式（标准版）
- **电源需求**：电池（AA/AAA × 4，续航6-12个月）
- **功能**：
  - ✅ 正常供水软化
  - ✅ 智能显示（水质、盐量、流量）
  - ✅ 再生提醒（LED/蜂鸣器）
  - ⚠️ 手动再生（电池电量不足时）
  - ✅ 电动辅助再生（电池电量充足时）
- **适用场景**：有电池、希望智能提醒、不想布线

#### 模式C：全功能模式（豪华版）
- **电源需求**：12V DC电源适配器（或电池+太阳能辅助）
- **功能**：
  - ✅ 所有模式B功能
  - ✅ 全自动再生（定时/定量）
  - ✅ IoT连接（APP监控、远程设置）
  - ✅ 数据记录（用水统计、盐耗分析）
  - ✅ 停电记忆（EEPROM存储设置）
- **适用场景**：新房装修、智能家庭、高端用户

## 3. 核心技术组件设计

### 3.1 机械主控阀（核心层）

#### 设计要求
- **永不依赖电力**：纯机械结构，水压驱动
- **可靠性**：设计寿命10年+，免维护
- **紧凑性**：适合小型化（台面/壁挂）

#### 技术方案：水力驱动多路阀

**工作原理**：
```
正常供水模式：
进水 → 主控阀 → 树脂罐（自上而下） → 出水（软化完成）

再生模式（手动）：
1. 反洗：进水 → 主控阀 → 树脂罐（自下而上） → 排污
2. 吸盐：射流器利用水压产生负压 → 盐箱吸盐水 → 树脂罐 → 排污
3. 正洗：进水 → 主控阀 → 树脂罐（自上而下） → 排污
4. 盐箱补水：进水 → 盐箱（补充再生用水）
```

**关键组件**：
1. **水力驱动轮**：利用水压旋转阀芯（类似水表原理）
2. **凸轮机构**：控制阀门切换顺序
3. **手动手柄**：用户手动触发再生（或跳过自动再生）
4. **定时机构**（可选）：利用水压驱动的齿轮计时器（类似老式洗衣机定时器）

**技术来源**：
- **Runxin（润新）**：中国品牌，专业生产水力驱动软化阀
- **Fleck**：美国品牌，工业级水力驱动阀
- **参考产品**：Runxin F56 系列（手动/定时混合型）

#### 创新设计：可切换自动/手动再生

**设计思路**：
```
默认状态：机械阀按设定时间/流量自动再生（水力驱动定时器）
用户干预：按下"手动再生"按钮 → 立即启动再生周期
用户关闭自动：切换到"纯手动模式" → 只有用户操作才再生
```

**实现方式**：
- 机械式模式选择开关（类似老式洗衣机的"标准/强力/手动"选择）
- 无需电力，纯机械结构

### 3.2 智能模块（智能层）

#### 设计要求
- **低功耗**：电池供电，续航6-12个月
- **模块化**：可随时拆卸，不影响核心功能
- **防水**：IPX4以上（浴室环境）

#### 技术方案：电池供电的智能显示屏

**核心组件**：
1. **低功耗MCU**：STM32L系列或ESP32（深度睡眠模式）
2. **显示屏**：段码LCD（低功耗，阳光下可见）
3. **传感器**：
   - TDS传感器（检测进水/出水硬度）
   - 流量传感器（霍尔效应，低功耗）
   - 盐量检测（电极式或重量传感器）
4. **电源**：AA电池 × 4（或锂电池 + 太阳能辅助）

**显示内容**：
```
┌─────────────────────┐
│  KES Soft Water     │
│                     │
│  出水硬度: 15 ppm   │ ← TDS传感器
│  剩余盐量: 70%      │ ← 盐量检测
│  今日用水: 200L     │ ← 流量统计
│  下次再生: 3天后    │ ← 计时器
│                     │
│  [模式] [设置] [再生]│ ← 按钮
└─────────────────────┘
```

**智能功能**：
1. **再生提醒**：
   - LED闪烁 + 蜂鸣器（可关闭）
   - 显示"需要再生"提示
   - 用户可以选择立即手动再生或延迟

2. **用水统计**：
   - 记录每日/每月用水量
   - 计算盐耗（g/L）
   - 预估下次再生时间

3. **水质监控**：
   - 实时显示出水硬度
   - 硬度异常报警（树脂失效？）

4. **IoT连接**（豪华版）：
   - WiFi/蓝牙模块
   - APP远程监控
   - 云端数据分析

#### 创新设计：磁吸式智能模块

**设计思路**：
- 智能模块通过磁力吸附在软水机外壳上
- 内部线圈感应取电（类似无线充电）或接触式导电
- 用户可以随时拆下智能模块，软水机仍能正常工作

**好处**：
- 智能模块损坏/故障不影响基本功能
- 用户可以选择不购买智能模块（降低成本）
- 升级方便（旧机器加装智能模块）

### 3.3 动力辅助模块（动力层）

#### 设计要求
- **可选电源**：用户可以选配电源适配器
- **停电兼容**：有电源时用电源，停电时自动切换电池
- **安全**：低压供电（12V DC），防水设计

#### 技术方案：电动再生辅助

**核心组件**：
1. **微型电机**：12V DC，低扭矩（只需驱动阀门）
2. **减速齿轮箱**：增大扭矩，降低速度
3. **电磁阀门**：自动切换水流（再生时用）
4. **电源管理**：
   - 优先使用适配器电源
   - 停电时自动切换电池
   - 电池充电管理（如果有可充电电池）

**工作流程**：
```
用户选择"自动再生"模式：
1. 达到再生条件（时间/流量）
2. 电机驱动阀门切换到再生位置
3. 控制器按程序执行再生步骤（反洗→吸盐→正洗→补水）
4. 再生完成，电机驱动阀门切换回供水位置
5. 记录再生时间和盐耗
```

**手动覆盖**：
- 任何时候，用户可以手动操作再生手柄
- 手动操作优先级高于电动操作
- 手动操作时，电机会自动断开（离合器设计）

**布局**：

```
        │   智能显示屏         │ ← 可选，磁吸式
        ├─────────────────────┤
        │   机械主控阀         │ ← 手动再生手柄在此
```

**安装**：
- 放置于浴室台面
- 进水/出水通过软管连接（类似洗衣机）
- 可选电源适配器（12V DC）

### 4.2 壁挂式设计（Wall-Mount）

**尺寸**：50cm (H) × 30cm (W) × 15cm (D)（超薄）

**布局**：
```
    ┌─────────────────────┐
    │ 智能显示屏           │ ← 可选
    ├─────────────────────┤
    │ 机械主控阀           │
    │ (手动再生手柄)        │
    ├─────────────────────┤
    │ 树脂罐   │  盐箱      │ ← 左右分布，节省厚度
    │ (高40cm) │  (5kg)    │
    └─────────────────────┘
         ↑ 壁挂孔
```

**安装**：
- 挂墙安装（膨胀螺丝或免打孔胶）
- 进水/出水墙内预埋或明管
- 可选电源插座（高度1.5m）

**创新设计**：树脂罐和盐箱分离

**好处**：
- 适应狭小空间（盐箱可以放在柜子里）
- 树脂罐壁挂，盐箱落地
- 软管连接（2-3米）

**布局**：
```
树脂罐单元（壁挂）          盐箱单元（落地）
┌──────────────┐          ┌──────────────┐
│ 智能显示屏    │          │              │
```

## 5. 技术可行性分析

### 5.1 机械阀技术成熟度

**现有技术**：
- ✅ **Runxin（润新）**：中国品牌，水力驱动软化阀成熟产品
  - 型号：F56A（手动）、F56B（定时）、F56C（电动）
  - 价格：US$50-150（阀门本身）
  - 可靠性：10年+寿命

- ✅ **Fleck**：美国品牌，工业级阀门
  - 型号：5600（电动）、1500（手动）
  - 价格：US$200-500
  - 可靠性：工业级

**结论**：机械阀技术成熟，可直接采购或定制

### 5.2 智能模块技术可行性

**低功耗设计**：
- ✅ STM32L系列：待机电流 < 10µA
- ✅ 段码LCD：工作电流 < 10µA
- ✅ 传感器：只在测量时通电（每秒测量一次，每次10ms）
- **续航计算**：
  - 4×AA电池（2000mAh each）= 8000mAh
  - 平均电流：100µA（待机）+ 10mA（测量，每天1分钟）= ~10.007mA
  - 续航：8000mAh / 10.007mA ≈ 800小时 ≈ **33天**？

**问题**：上述计算有误，重新计算：
  - 待机时间：99.9%的时间（每天14382分钟）
  - 测量时间：0.1%的时间（每天1分钟 = 60秒）
  - 待机电流：10µA = 0.01mA
  - 测量电流：10mA
  - 平均电流 = (14382 × 0.01mA + 60 × 10mA) / 1440分钟
              = (143.82 + 600) / 1440
              = 743.82 / 1440
              = 0.516mA
  - 续航 = 8000mAh / 0.516mA = 15504小时 = **1.77年**

**优化**：
- 使用更大容量电池（18650锂电池，2000mAh × 2 = 4000mAh each）
- 太阳能辅助充电（显示屏上方加装太阳能板）
- **目标续航**：2-3年

**结论**：低功耗智能模块技术可行，续航可达2-3年

## 6. 竞争优势分析

### 6.1 vs 传统电动软水机

| 维度 | KES混合动力 | 安吉尔/ A.O.史密斯 |
|------|------------|-------------------|
| **电源需求** | 可选 | 必须 |
| **停电影响** | 无 | 无法使用 |
| **安装复杂度** | 低（无需布线） | 高（需电源插座） |
| **可靠性** | 高（机械为主） | 中（电子元件） |
| **价格** | 低-中 | 高 |
| **智能功能** | 可选 | 标配 |

**差异化定位**：
- "最可靠的软水器" - 停电也能用
- "安装最简单的软水器" - 无需电工
- "最灵活的软水器" - 用户选择功能等级

### 6.2 vs 无电手动软水机

| 维度 | KES混合动力 | 传统手动软水机 |
|------|------------|----------------|
| **智能功能** | 可选 | 无 |
| **再生方式** | 手动/自动可选 | 只能手动 |
| **用户友好度** | 高（有提醒） | 低（易忘记再生） |
| **价格** | 中-高 | 低 |
| **现代化** | 是 | 否 |

**差异化定位**：
- "传统可靠 + 现代便利"
- 吸引年轻用户（喜欢智能功能但重视可靠性）

### 6.3 vs 淋浴过滤器

**完全不在一个档次**：
- KES混合动力是真正的软水器（离子交换）
- 淋浴过滤器只是过滤+部分软化
- 价格差异大（KES ~¥2,500 vs 淋浴过滤器 ~¥200）
- **但KES可以定位为"升级方案"**：从淋浴过滤器升级到真正的软水器

## 9. 营销策略建议

### 9.1 核心价值传播

**主标语**：
- "永远有软水用 - 即使停电"
- "智能，但不依赖电"
- "你选择功能，不是功能选择你"

**支撑点**：
1. **可靠性**：机械为主，10年寿命
2. **灵活性**：三种模式，用户选择
3. **安装简单**：无需电工，自己也能装
4. **性价比**：基础版价格低，升级方便

### 9.2 目标市场细分

#### 市场A：改造市场（无电源）
- **痛点**：浴室没有电源插座，无法装电动软水机
- **解决方案**：KES基础版/标准版（电池供电）
- **营销渠道**：装修公司、水电工、DIY社区

#### 市场B：停电频繁地区
- **痛点**：停电后软水机不能用，硬水问题复发
- **解决方案**：KES混合动力（机械阀保障）
- **营销渠道**：地方经销商、电力公司有合作

#### 市场C：智能家庭用户
- **痛点**：希望智能功能，但担心可靠性
- **解决方案**：KES豪华版（全功能 + 机械备份）
- **营销渠道**：智能家居集成商、线上旗舰店

#### 市场D：浴缸场景用户（你的洞察！）
- **痛点**：想要浴缸软水，但不想大规模改造
- **解决方案**：KES台面式（放置于浴缸旁，软管连接）
- **营销渠道**：卫浴店、装修博客、小红书

### 9.3 产品命名建议

**系列名**：KES Hybrid 系列

**型号**：
- KES Hybrid S1（基础版 - 纯机械）
- KES Hybrid S2（标准版 - +智能显示）
- KES Hybrid S3（豪华版 - +全自动 + IoT）

**卖点包装**：
- "机械的可靠，智能的便利"
- "三种模式，随你选择"
- "停电？没关系，软水继续"

## 11. 结论

**混合动力小型软水器是一个有价值的产品方向**，原因如下：

1. **技术可行**：机械阀技术成熟，智能模块低功耗可行
2. **市场空白**：目前市场上缺少"无电也能用"的小型软水器
3. **差异化明显**：可靠性 + 灵活性是强卖点
4. **成本可控**：量产成本可降至US$150-200，零售价有竞争力
5. **用户价值高**：解决停电焦虑、安装困难、选择有限等痛点

**建议**：
- ✅ 推进到概念验证阶段（采购阀门，搭建原型）
- ✅ 深入用户调研（验证"停电焦虑"是否真实存在）
- ✅ 联系Runxin等阀门厂家（探讨定制可能性）
