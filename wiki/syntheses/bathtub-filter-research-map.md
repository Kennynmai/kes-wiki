## Framing
这页是 bathtub filter 主题的可复用 research map（研究地图）。它的作用是把未来的资料采集、source reading 与 write-back 结构化，让这个品类能被系统性探索，而不是只靠零散笔记推进。

## Findings
### 首先需要建立的内容
1. demand existence（需求是否存在）
2. user pain segmentation（用户痛点分层）
3. product-form reality（产品形态现实）
4. claims / compliance constraints（宣称 / 合规约束）
5. platform-commercial fit（平台商业匹配）

### 可能的研究轨道（Likely research tracks）
#### 0. Academic-search infrastructure track
问题：
- 未来的 evidence work 应由什么 metadata / search stack 支撑？
- manuscript / repository access depth 应如何记录？
- 一个产品主题在什么情况下值得启用专门 academic workflow？

交付物：
- academic-search workflow
- access-state vocabulary
- evidence-access audit habit

#### 1. Search-intent track
问题：
- 用户是否会直接搜索 “bathtub filter”？
- 更常见的相邻词是什么？
- 需求究竟是围绕 babies、skin、chlorine、hard water、eczema、wellness，还是 odor？

交付物：
- keyword map
- intent clusters
- naming recommendations

#### 2. Product-form track
问题：
- 市场上实际存在什么 forms？
- 很多 “filters” 是否其实更像 treatment accessories 或 media products，而不是真正的 filtration devices？
- 哪些 form factors 在运营上最容易解释、最容易支持？

交付物：
- form-factor taxonomy
- install / compatibility matrix
- complexity / risk notes

#### 3. Consumer-pain track
问题：
- reviews 中反复出现的 customer problems 是什么？
- 哪些 pains 更偏 emotional，哪些更偏 functional？
- 哪些 customer segments 最活跃？

可能的 segment buckets：
- baby / children
- sensitive skin / eczema
- hard-water households
- chlorine-smell / comfort seekers
- wellness / premium bath users

#### 4. Claims / trust track
问题：
- 头部品牌都在用哪些 claims？
- 哪些 claims 看起来风险高或难以验证？
- 市场上在 signal 哪些 certifications 或 test standards？

交付物：
- claims library
- red-flag claims list
- certification gap notes

#### 5. Commercial track
问题：
- 市场上有哪些 price bands？
- 是否存在 replacement-cartridge economics？
- 复购是否有意义？
- 低评分的典型原因是什么？

交付物：
- competitor price map
- margin hypothesis notes
- failure-mode / return-risk map

#### 6. Patent / technical-route track
问题：
- 哪些 product architectures 已经在 prior art 中存在？
- 这个品类主要是 bathtub-specific architecture，还是 shower-filter adjacency？
- 哪些 claims 在技术上更可信，哪些更像 marketing-driven？
- 某些 attachment 或 cartridge formats 是否有明显 IP crowding？

交付物：
- patent / technical landscape
- route-type taxonomy
- future assignee / family map

## Fact vs inference
### 目前已建立的事实
- KES 正在评估当前核心品类之外的 category adjacency。
- Bathtub filter 已被纳入 KES 持续追踪的研究主题。

### 当前仅属推论（inferences）
- 该品类可能很依赖教育成本。
- 相比普通 hardware，它可能带有更高的 claims / compliance risk。
- 相比 commodity hardware，它可能更依赖 premium / problem-solution storytelling。

这些都需要真正的市场证据，才能再写回到更正式的 surfaces。

## Tradeoffs / unknowns
- 有吸引力的问题解决叙事 vs. 更高的信任负担
- premium positioning 潜力 vs. certification / claims constraints
- accessory 的简洁性 vs. compatibility 的复杂性
- repeat-filter economics vs. support 与 complaint load

## Implications
如果这个品类被证明真实可做，它可能会为 KES 打开一个更广的研究支线：
- home water quality accessories
- skin-sensitive bathroom products
- premium bathing accessories
- filtration-adjacent bathroom hardware

如果最终判断较弱，它仍然可能是一个有价值的 demand-sensing case study（需求感知案例），帮助 KES 形成一套评估 health- / wellness-adjacent bathroom categories 的方法。
