### 2.2 工作原理

```
[硬水进水]
     ↓
[离子交换树脂罐]
  Ca²⁺/Mg²⁺ → 树脂吸附
  Na⁺ → 释放到水中
     ↓
[软水出水] → 淋浴
```

**再生过程**（手动）：
```
1. 拆下设备
2. 准备盐水（500g盐 + 2L水）
3. 倒入再生口
4. 等待4小时（Na⁺置换Ca²⁺/Mg²⁺）
5. 反冲15分钟（去除多余盐分）
6. 重新安装
```

## 五、技术可行性分析

### 5.1 柠檬酸再生 —— 技术可行性

#### **化学原理（为什么柠檬酸不能直接替代盐？）**

**传统盐再生的化学原理**：

```
树脂饱和过程（工作阶段）：
2R-Na + Ca²⁺ → R₂-Ca + 2Na⁺
2R-Na + Mg²⁺ → R₂-Mg + 2Na⁺

再生过程（用NaCl）：
R₂-Ca + 2Na⁺ → 2R-Na + Ca²⁺
R₂-Mg + 2Na⁺ → 2R-Na + Mg²⁺
```

**问题：柠檬酸能这样做吗？**

❌ **不能直接替代**，原因：

1. **柠檬酸是弱有机酸**（pKa₁=3.13, pKa₂=4.76, pKa₃=6.40）
2. 如果用于再生阳离子树脂，会将树脂转化为 **H⁺形式**（酸性），而不是Na⁺形式
3. 当H⁺型树脂接触硬水时：
   ```
   R-H + Ca²⁺ → R₂-Ca + 2H⁺
   ```
   - 释放的H⁺会让水变**酸性**（pH可能降到5-6）
   - 对皮肤和头发**有害**

---

#### **柠檬酸的真实用途（不是再生，是清洗）**

**正确用途**：树脂清洗剂（去除污染）

| 污染物 | 为什么柠檬酸有效？ | 使用场景 |
|---------|-------------------|---------|
| **铁污染**（Fe³⁺） | 柠檬酸与Fe³⁺形成稳定络合物 | 井水/铁含量高的地区 |
| **锰污染**（Mn²⁺） | 类似铁，形成络合物 | 地下水 |
| **有机物污染** | 柠檬酸的羧基可以溶解有机物 | 地表水/水库水 |
| **碳酸钙结垢** | 酸性溶解CaCO₃ | 树脂床结垢 |

**操作方式**：
```
步骤1：用10-15%柠檬酸溶液浸泡树脂4-8小时
步骤2：慢速反洗，去除络合的Fe³⁺/Mn²⁺
步骤3：用盐水（NaCl）正常再生，恢复Na⁺型
```

**关键**：柠檬酸是**清洗剂**，不是**再生剂**。再生还是需要盐！

---

#### **真正实现"免盐"的技术方案**

如果你想要**真正的免盐再生**，有几种技术路线：

##### **方案A：弱酸阳离子树脂（WAC）+ 酸再生**

| 维度 | 强酸阳离子树脂（SAC） | 弱酸阳离子树脂（WAC） |
|------|----------------------|----------------------|
| **再生剂** | NaCl（盐） | 酸（HCl、H₂SO₄）或柠檬酸 |
| **交换容量** | 高（2.0-2.4 eq/L） | 中（1.5-2.0 eq/L） |
| **再生效率** | 中等 | **高（用酸量少）** |
| **出水pH** | 中性（Na⁺型） | **酸性（H⁺型）** ⚠️ |
| **适合淋浴？** | ✅ 是 | ❌ 否（出水酸性） |

**结论**：WAC + 柠檬酸再生，**出水是酸性的**，不适合淋浴！

---

##### **方案B：螯合树脂 + 酸再生**

**原理**：使用**螯合树脂**（如Amberlite IRC748，专用于Ca²⁺/Mg²⁺）

| 维度 | 说明 |
|------|------|
| **树脂类型** | 亚氨基二乙酸型（IDA）或氨基膦酸型 |
| **选择性** | 对Ca²⁺/Mg²⁺选择性极高 |
| **再生剂** | HCl、H₂SO₄、或柠檬酸 |
| **再生效率** | 高（用酸量比NaCl少50-70%） |
| **出水pH** | 酸性（H⁺型）⚠️ |

**解决方案**：需要**后置中和**（加碱性介质），增加复杂度和成本。

---

##### **方案C：Template Assisted Crystallization (TAC) —— 真正的免盐方案**

**原理**：不是去除钙镁离子，而是改变其结晶形态，使其**不结垢**

| 维度 | 离子交换（盐再生） | TAC（免盐） |
|------|-------------------|--------------|
| **原理** | 去除Ca²⁺/Mg²⁺ | 改变结晶形态（球状晶体） |
| **需要盐？** | ✅ 是 | ❌ 否 |
| **真正软化？** | ✅ 是（去除硬度） | ❌ 否（保留硬度，但不结垢） |
| **对皮肤/头发好处** | ✅ 明显 | ⚠️ 有限 |
| **寿命** | 10年+ | 3-5年 |
| **成本** | 中 | 高（TAC介质贵） |

**结论**：TAC不是真正的"软水"，只是"阻垢"。对皮肤/头发的好处**不如真正软水**。

---

##### **方案D：柠檬酸在线添加（最实用的"免盐"方案）**

**原理**：不是再生树脂，而是在淋浴水中**直接添加柠檬酸**，络合Ca²⁺/Mg²⁺

**系统设计**：

```
[进水] → [柠檬酸添加泵] → [混合腔] → [淋浴]
                            ↓
                    [CaCitrate络合物]（不结垢）
```

| 维度 | 说明 |
|------|------|
| **优点** | 真正免盐，无需树脂，成本低 |
| **缺点** | 需要持续添加柠檬酸（运行成本高） |
| **柠檬酸用量** | 根据硬度：5-20g/100L水 |
| **皮肤安全性** | ✅ 食品级柠檬酸，安全 |
| **设备复杂度** | 中（需要计量泵+控制器） |

**适合KES产品吗？**
- ❌ **不适合便携式淋浴软水器**（需要持续添加柠檬酸）
- ✅ **适合全屋系统**（可以集中添加）

---

#### **推荐方案（KES产品）：双级再生**

##### **2. 两种控制方式**

| 方式 | 原理 | 优点 | 缺点 | 推荐？ |
|------|------|------|------|--------|
| **时间型** | 按设定时间周期再生（如每7天） | 简单，成本低 | 不管用水量多少都再生（浪费盐） | ⚠️ 不推荐 |
| **流量型** | 按实际用水量触发再生 | 省盐，科学 | 需要流量计，成本稍高 | ✅ **推荐** |

##### **3. 流量型再生的具体原理**

###### **核心组件**：

```
[进水] → [A] 流量计 → [主控板] → [B] 电动阀门
           ↓           ↓
      [累计水量]  [触发判断]  [切换水路]
```

###### **工作流程**：

| 步骤 | 说明 | 示例 |
|------|------|------|
| 1. 初始设置 | 用户设置进水硬度（如15 gpg） | 系统计算：1L水 = 15mg/L × 0.2 = 3 grains/L |
| 2. 用水计量 | 流量计实时计量用水量 | 每次淋浴10分钟 = 约80L |
| 3. 累计计算 | MCU累计计算已处理的总硬度 | 80L × 3 grains/L = 240 grains |
| 4. 剩余容量 | 树脂总容量 - 已处理量 | 10,000 grains - 240 = 9,760 grains剩余 |
| 5. 触发再生 | 剩余容量 < 10%时触发 | 剩余<1,000 grains时，凌晨自动再生 |

###### **为什么用流量型？**

假设用户A和B：

| 用户 | 每天淋浴 | 树脂容量 | 时间型再生 | 流量型再生 |
|------|-----------|-----------|--------------|--------------|
| A（单人） | 1次，7分钟 | 3000L | 每7天再生（浪费） | 每14天再生（省盐） |
| B（4口之家） | 4次，10分钟 | 3000L | 每7天再生（不够用） | 每3-4天再生（刚好） |

**结论**：流量型更科学，省盐30-50%

##### **3. 再生步骤详细分解**

###### **完整再生周期（50分钟）**：

| 步骤 | 时间 | 阀门状态 | 水泵状态 | 目的 |
|------|------|---------|---------|------|
| **1. 反冲洗** | 0-10分钟 | 进水关，排水开 | 关 | 冲起树脂床，去除悬浮物 |
| **2. 吸盐** | 10-30分钟 | 进水关，排水开 | 开 | 盐水通过树脂，Na⁺置换Ca²⁺/Mg²⁺ |
| **3. 慢洗** | 30-40分钟 | 进水关，排水开 | 关 | 低速清水冲洗，去除多余盐分 |
| **4. 快洗** | 40-50分钟 | 进水关，排水开 | 关 | 高速清水冲洗，恢复树脂床 |
| **5. 恢复** | 50分钟 | 进水开，排水关 | 关 | 切换回正常工作模式 |

**关键设计要点**：
1. **反冲洗很重要**：树脂床会压实，反冲可以松动树脂，提高再生效率
2. **吸盐时间要足够**：让Na⁺充分置换Ca²⁺/Mg²⁺（至少20分钟）
3. **清洗要彻底**：避免出水含盐量过高

##### **2. 再生执行流程（状态机）**

```cpp
enum RegenState {
  IDLE,          // 待机
  BACKWASH,      // 反冲洗
  BRINE_DRAW,    // 吸盐
  SLOW_RINSE,    // 慢洗
  FAST_RINSE,    // 快洗
  RETURN_TO_SERVICE, // 恢复
  ERROR          // 故障
};

void startRegeneration() {
  currentState = BACKWASH;
  startTime = millis();
  
  // 关闭进水阀，打开排水阀
  closeInletValve();
  openDrainValve();
  
  logToApp("再生开始：反冲洗阶段");
}

void updateRegeneration() {
  unsigned long elapsed = (millis() - startTime) / 1000 / 60; // 分钟
  
  switch (currentState) {
    case BACKWASH:
      if (elapsed >= 10) {
        currentState = BRINE_DRAW;
        startTime = millis();
        startBrinePump();
        logToApp("再生进度20%：吸盐阶段");
      }
      break;
      
    case BRINE_DRAW:
      if (elapsed >= 30) {
        currentState = SLOW_RINSE;
        startTime = millis();
        stopBrinePump();
        logToApp("再生进度60%：慢洗阶段");
      }
      break;
      
    case SLOW_RINSE:
      if (elapsed >= 40) {
        currentState = FAST_RINSE;
        startTime = millis();
        logToApp("再生进度80%：快洗阶段");
      }
      break;
      
    case FAST_RINSE:
      if (elapsed >= 50) {
        currentState = RETURN_TO_SERVICE;
        openInletValve();
        closeDrainValve();
        resetFlowMeter();
        logToApp("再生完成：设备已恢复正常");
        sendAppNotification("再生完成，设备已恢复正常");
      }
      break;
  }
}
```

### 8.2 成本控制（价格敏感市场）

**目标成本结构（中国版）**：

**降本路径**：

### 8.3 再生体验（用户痛点）

**ShowerStick的再生流程（太复杂）**：
1. 拆下设备
2. 准备盐水（500g盐+2L水）
3. 倒入再生口
4. 等待4小时
5. 反冲15分钟
6. 重新安装

**KES优化方案**：
- 🔧 **快拆式盐仓**：类似胶囊咖啡机，换盐就像换胶囊
- ⏱️ **定时再生**：App设置，凌晨自动再生（需电动阀门）
- 💧 **自反冲设计**：再生时自动反冲，无需手动操作
- 🍋 **免盐方案**（中国版）：用柠檬酸再生，更安全
