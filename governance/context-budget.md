# Context Budget — KES Company Wiki

Use progressive disclosure with explicit context budgets.

## L0 — Task Context (~200 tokens)
Current objective, business domain, task type, and stakes.

## L1 — Index Layer (~1–2K)
`index.md`, dashboards, page titles, one-line descriptions, recent updates.

## L2 — Relevant Maintained Pages (~2–5K)
Platform, market, product, project, decision, playbook, synthesis, risk, and source-summary pages.

## L3 — Full Source Layer (~5–20K)
Reports, raw meeting notes, transcripts, policy documents, data explanations, source passages required for verification.

## Reading discipline
Default progression:
L0 -> L1 -> L2 -> L3

Do not read full documents by default.
Do not make high-stakes claims from L1 alone.
