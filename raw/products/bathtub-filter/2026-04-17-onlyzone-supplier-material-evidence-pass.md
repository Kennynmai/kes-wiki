# Bathtub Filter — Onlyzone Supplier Material Evidence Pass (2026-04-17)

## Purpose
Capture of **first-party supplier-side certifications and test reports** from **淄博宗立 / Zibo Onlyzone Water Treatment Technology Co., Ltd** — the media supplier for KES's Version A bathtub filter (KDF55 + CaSO₃).

These reports sit at the **material level**, not the finished-KES-product level. Their main roles:
1. Validate the math on Version A's life model (the 40g CaSO₃ reference test is exactly what η=0.9 was scaled from — see math check below)
2. Document the evidence stack KES can cite for supplier credibility (NSF/ANSI 42 listing, EU food-contact, antimicrobial, Pb removal)
3. Preserve the critical distinction between "supplier-proven media" and "KES-proven finished product"

## Math validation: η = 0.9 scaling from supplier report → Version A life model

Version A uses scaling factor **3.25 × η(0.9) = 2.925×** to project 40g supplier test → 130g KES stack. Interpolation rule: "last ≥ threshold point → first confidently < threshold point, linear interpolated; ignore non-monotonic reversion, use the *latest* ≥ threshold point".

| Threshold | Supplier last ≥ point | Supplier first < point | Linear interp (L) | × 2.925 | Version A doc says | Match |
|---|---|---|---|---|---|---|
| ≥99% | 1020 L @ 99.2% | 1870 L @ 98.2% | 1020 + 0.2×(1870−1020) = **1190** | **3,481 L** | 3,481 L | ✅ EXACT |
| ≥95% | 5780 L @ 95.5% | 6460 L @ 94.2% | 5780 + 0.385×(6460−5780) = **6042** | **17,673 L** | 17,672 L | ✅ EXACT (rounding) |
| ≥90% | 6460 L @ 94.2% | 7650 L @ 82.3% | 6460 + 0.353×(7650−6460) = **6880** | **20,124 L** | 20,124 L | ✅ EXACT |
| ≥50% | 8670 L @ 52.9% | (not observed) | 8670 + extrapolation 37 = **8707** | **25,467 L** | 25,467 L | ✅ EXACT |

**Conclusion**: the life model math is internally consistent and traceable. η = 0.9 remains an **assumed** efficiency factor (no third-party finished-product verification), but the input data and the scaling rule now have full audit trail.

**Known gap (still open)**: the supplier test used **0.5–1 mm** CaSO₃ particles; Version A production uses **3–4 mm** particles. Larger particles = less surface area per gram = **slower** reaction kinetics. The η=0.9 factor may partially capture this, but the particle-size delta is a non-trivial variable that should be measured on the actual production KES stack. This is currently encapsulated inside "η=0.9" without visibility.
